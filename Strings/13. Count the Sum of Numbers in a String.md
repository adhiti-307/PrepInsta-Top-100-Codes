## 🧩 Problem: Count the Sum of Numbers in a String

### 📄 Problem Statement
Write a Java program to find the sum of all numbers present in a given string.

### 📥 Input
A string `str` containing alphabets and digits.

### 📤 Output
Print the sum of all numeric values present in the string.

### 💡 Example

**Input:**
```
a1b2c3
```

**Output:**
```
6
```

**Input:**
```
12abc20yz3
```

**Output:**
```
35
```

### 🧠 Approach
1. Traverse each character of the string.  
2. If the character is a digit, accumulate it to form numbers (handle multi-digit numbers).  
3. When a non-digit is encountered, add the formed number to the sum.  
4. Continue until the end of the string and print the total sum.  

---

### 💻 Java Code

#### 🔹 Method 1: Handling Multi-digit Numbers
```java
import java.util.*;

public class Main {
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);
        String str = sc.nextLine();

        int sum = 0, num = 0;

        for(int i = 0; i < str.length(); i++){
            char ch = str.charAt(i);

            if(Character.isDigit(ch)){
                num = num * 10 + (ch - '0');
            } else {
                sum += num;
                num = 0;
            }
        }

        sum += num; // last number

        System.out.println(sum);
    }
}
```

#### 🔹 Method 2: Sum of Individual Digits Only
```java
import java.util.*;

public class Main {
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);
        String str = sc.nextLine();

        int sum = 0;

        for(char ch : str.toCharArray()){
            if(Character.isDigit(ch)){
                sum += ch - '0';
            }
        }

        System.out.println(sum);
    }
}
```
