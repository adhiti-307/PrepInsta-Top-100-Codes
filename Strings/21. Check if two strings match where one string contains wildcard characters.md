# 22. Check if Two Strings Match with Wildcards

## 📌 Problem Statement

Write a program to check whether two strings match, where one of the strings may contain wildcard characters.

Wildcards:

* `?` → matches **exactly one character**
* `*` → matches **zero or more characters**

---

## 📝 Input Format

* First line: String `text`
* Second line: Pattern `pattern` (may contain `?` and `*`)

---

## 📤 Output Format

* Print `true` if the strings match
* Otherwise print `false`

---

## 🔍 Constraints

* Pattern may contain lowercase/uppercase characters, `?`, and `*`
* Matching should cover the **entire string**

---

## 💡 Approach (Dynamic Programming)

Let:

```text
dp[i][j] = true if first i characters of text match
           first j characters of pattern
```

---

### 🔁 Rules

1. **If characters match OR pattern has `?`**

```text
dp[i][j] = dp[i-1][j-1]
```

2. **If pattern has `*`**

```text
dp[i][j] = dp[i][j-1]   (treat * as empty)
        OR dp[i-1][j]   (match one more character)
```

3. **Else**

```text
dp[i][j] = false
```

---

## 💻 Java Code

```java
import java.util.*;

class Main {

    static boolean isMatch(String text, String pattern) {
        int n = text.length();
        int m = pattern.length();

        boolean[][] dp = new boolean[n + 1][m + 1];

        dp[0][0] = true;

        // Handle patterns like *, **, ***
        for (int j = 1; j <= m; j++) {
            if (pattern.charAt(j - 1) == '*') {
                dp[0][j] = dp[0][j - 1];
            }
        }

        for (int i = 1; i <= n; i++) {
            for (int j = 1; j <= m; j++) {

                char pc = pattern.charAt(j - 1);
                char tc = text.charAt(i - 1);

                if (pc == tc || pc == '?') {
                    dp[i][j] = dp[i - 1][j - 1];
                }
                else if (pc == '*') {
                    dp[i][j] = dp[i][j - 1] || dp[i - 1][j];
                }
                else {
                    dp[i][j] = false;
                }
            }
        }

        return dp[n][m];
    }

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        String text = sc.nextLine();
        String pattern = sc.nextLine();

        System.out.println(isMatch(text, pattern));
    }
}
```

---

## 🧪 Sample Input

```text
baaabab
ba*a?
```

---

## ✅ Sample Output

```text
true
```

---

## 🧠 Explanation

Pattern:

```text
ba*a?
```

Matches:

```text
b a a a b a b
| | *   | ?
```

* `*` → matches multiple characters
* `?` → matches one character

---

## 📊 Complexity Analysis

* Time Complexity: **O(n × m)**
* Space Complexity: **O(n × m)**

---

## 🔥 Key Points

* `?` → matches exactly one character
* `*` → matches zero or more characters
* Use DP to track all matching possibilities
* Must match the **entire string**, not partial
