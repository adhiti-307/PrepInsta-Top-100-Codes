# 17. Check if Two Strings are Anagrams (Java)

## 📌 Problem Statement

Given two strings, determine whether they are **anagrams** of each other.
Two strings are anagrams if they contain the **same characters with the same frequency**, possibly in a different order.

---

## 🧠 Approach 1: Using HashMap (Works for all characters)

### 🔹 Idea

* Count frequency of characters in the first string
* Decrease frequency using the second string
* If all values become `0`, strings are anagrams

---

### 💻 Java Code

```java
import java.util.*;

class Main{
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);
        String str1 = sc.nextLine();
        String str2 = sc.nextLine();

        // Length check
        if(str1.length() != str2.length()){
            System.out.println("Not Anagram");
            return;
        }

        HashMap<Character,Integer> map = new HashMap<>();

        // Count frequency of str1
        for(char ch : str1.toCharArray()){
            map.put(ch, map.getOrDefault(ch, 0) + 1);
        }

        // Subtract frequency using str2
        for(char ch : str2.toCharArray()){
            map.put(ch, map.getOrDefault(ch, 0) - 1);
        }

        // Check all values are zero
        for(Map.Entry<Character,Integer> e : map.entrySet()){
            if(e.getValue() != 0){
                System.out.println("Not Anagram");
                return;
            }
        }

        System.out.println("Anagram");
    }
}
```

---

### ✅ Example

**Input:**

```
listen
silent
```

**Output:**

```
Anagram
```

## ⚡ Approach 2: Using Frequency Array (Optimized)

### 🔹 Idea

* Use an array of size 26 (for lowercase letters)
* Increment for first string, decrement for second
* If all values are zero → anagram

---

### 💻 Java Code

```java
import java.util.*;

class Main{
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);
        String str1 = sc.nextLine();
        String str2 = sc.nextLine();

        if(str1.length() != str2.length()){
            System.out.println("Not Anagram");
            return;
        }

        int[] freq = new int[26];

        for(char c : str1.toCharArray()){
            freq[c - 'a']++;
        }

        for(char c : str2.toCharArray()){
            freq[c - 'a']--;
        }

        for(int f : freq){
            if(f != 0){
                System.out.println("Not Anagram");
                return;
            }
        }

        System.out.println("Anagram");
    }
}
```

---

### ✅ Example

**Input:**

```
triangle
integral
```

**Output:**

```
Anagram
```

---

## 📊 Complexity Analysis

| Approach | Time Complexity | Space Complexity |
| -------- | --------------- | ---------------- |
| HashMap  | O(n)            | O(n)             |
| Array    | O(n)            | O(1)             |

---

## 🔥 Key Takeaways

* Use **HashMap** when:

  * Characters may include uppercase, symbols, or Unicode
* Use **array approach** when:

  * Input is limited to lowercase letters → faster and more memory efficient
* Always perform a **length check first**
