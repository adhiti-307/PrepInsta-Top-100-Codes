## 🧩 Problem: Check Whether a Character is an Alphabet or Not

### 📄 Problem Statement
Write a Java program to check whether a given character is an alphabet or not.

### 📥 Input
A single character `ch`.

### 📤 Output
Print `"Alphabet"` if the character is a letter (A–Z or a–z), otherwise print `"Not Alphabet"`.

### 💡 Example

**Input:**
```
A
```

**Output:**
```
Alphabet
```

**Input:**
```
9
```

**Output:**
```
Not Alphabet
```

### 🧠 Approach 1
1. Check if the character lies between `'a'` to `'z'` or `'A'` to `'Z'`.  
2. If yes → it is an alphabet, else → not an alphabet.
   
### 🧠 Approach 2
1. Use `Character.isLetter(ch)` method.  
2. If it returns `true` → alphabet, else → not alphabet.  

---

### 💻 Java Code Method 1 
```java
import java.util.*;

public class Main {
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);
        char ch = sc.next().charAt(0);

        if((ch >= 'a' && ch <= 'z') || (ch >= 'A' && ch <= 'Z'))
            System.out.println("Alphabet");
        else
            System.out.println("Not Alphabet");
    }
}
```

### 💻 Java Code Method 2
```java
import java.util.*;

public class Main {
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);
        char ch = sc.next().charAt(0);

        if(Character.isLetter(ch))
            System.out.println("Alphabet");
        else
            System.out.println("Not Alphabet");
    }
}
```
