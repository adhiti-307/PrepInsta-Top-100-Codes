## 🧩 Problem: Remove All Characters from String Except Alphabets

### 📄 Problem Statement
Write a Java program to remove all non-alphabet characters from a given string.

### 📥 Input
A string `str`.

### 📤 Output
Print the string containing only alphabet characters (A–Z, a–z).

### 💡 Example

**Input:**
```
a1b2c3
```

**Output:**
```
abc
```

**Input:**
```
Hello@123!
```

**Output:**
```
Hello
```

### 🧠 Approach
1. Traverse each character of the string.  
2. Check if the character is an alphabet.  
3. If yes, append it to the result string.  
4. Ignore all other characters.  

---

### 💻 Java Code

#### 🔹 Method 1: Using Character Check
```java
import java.util.*;

public class Main {
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);
        String str = sc.next();

        String result = "";

        for(char ch : str.toCharArray()){
            if((ch >= 'a' && ch <= 'z') || (ch >= 'A' && ch <= 'Z')){
                result += ch;
            }
        }

        System.out.println(result);
    }
}
```

#### 🔹 Method 2: Using Inbuilt Method
```java
import java.util.*;

public class Main {
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);
        String str = sc.next();

        String result = "";

        for(char ch : str.toCharArray()){
            if(Character.isLetter(ch)){
                result += ch;
            }
        }

        System.out.println(result);
    }
}
```

#### 🔹 Method 3: Using Inbuilt Method(2)
```java
import java.util.*;

public class Main {
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);
        String str = sc.next();

        String result = "";

        for(int i =0;i < str.length();i++){
            char ch = str.charAt(i);
            if(Character.isLetter(ch)){
                result += ch;
            }
        }

        System.out.println(result);
    }
}
```
