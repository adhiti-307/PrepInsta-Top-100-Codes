## 🧩 Problem: Find Length of a String

### 📄 Problem Statement
Write a Java program to find the length of a given string using both inbuilt method and without using any method.

### 📥 Input
A string `str`.

### 📤 Output
Print the length of the string.

### 💡 Example

**Input:**
```
hello
```

**Output:**
```
5
```

### 🧠 Approach
1. **Using Inbuilt Method:**  
   Use `str.length()` to directly get the length.

2. **Without Using Method:**  
   Traverse the string character by character and count manually.

---

### 💻 Java Code

#### 🔹 Method 1: Using Inbuilt Method
```java
import java.util.*;

public class Main {
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);
        String str = sc.next();

        System.out.println(str.length());
    }
}
```

---

#### 🔹 Method 2: Without Using Inbuilt Method
```java
import java.util.*;

public class Main {
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);
        String str = sc.next();

        int count = 0;

        for(char ch : str.toCharArray()){
            count++;
        }

        System.out.println(count);
    }
}
```
