## 🧩 Problem: Count the Number of Vowels in a String

### 📄 Problem Statement
Write a Java program to count the number of vowels present in a given string.

### 📥 Input
A string `str`.

### 📤 Output
Print the count of vowels in the string.

### 💡 Example

**Input:**
```
hello
```

**Output:**
```
2
```

**Input:**
```
PROGRAMMING
```

**Output:**
```
3
```

### 🧠 Approach
1. Traverse each character of the string.  
2. Convert each character to lowercase for uniform comparison.  
3. Check if the character is one of: `a, e, i, o, u`.  
4. Increment count whenever a vowel is found.  

---

### 💻 Java Code

#### 🔹 Method 1: Using Direct Comparison
```java
import java.util.*;

public class Main {
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);
        String str = sc.next();

        int count = 0;

        for(int i = 0; i < str.length(); i++){
            char ch = Character.toLowerCase(str.charAt(i));

            if(ch == 'a' || ch == 'e' || ch == 'i' || ch == 'o' || ch == 'u'){
                count++;
            }
        }

        System.out.println(count);
    }
}
```

---

#### 🔹 Method 2: Using String Contains
```java
import java.util.*;

public class Main {
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);
        String str = sc.next();

        int count = 0;
        String vowels = "aeiouAEIOU";

        for(char ch : str.toCharArray()){
            if(vowels.indexOf(ch) != -1){
                count++;
            }
        }

        System.out.println(count);
    }
}
```
