## 🧩 Problem: Capitalize the First and Last Character of Each Word

### 📄 Problem Statement
Write a Java program to capitalize the first and last character of each word in a given string.

### 📥 Input
A string `str`.

### 📤 Output
Print the modified string where the first and last characters of each word are capitalized.

### 💡 Example

**Input:**
```
hello world
```

**Output:**
```
HellO WorlD
```

**Input:**
```
java programming
```

**Output:**
```
JavA ProgramminG
```

### 🧠 Approach
1. Split the string into words using space.  
2. For each word:
   - Convert first character to uppercase  
   - Convert last character to uppercase  
3. Keep the middle part unchanged  
4. Join all words back into a string  

---

### 💻 Java Code

#### 🔹 Method 1: Using Split
```java
import java.util.*;

public class Main {
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);
        String str = sc.nextLine();

        String[] words = str.split(" ");
        String result = "";

        for(String word : words){
            int len = word.length();

            if(len == 1){
                result += Character.toUpperCase(word.charAt(0)) + " ";
            } else {
                char first = Character.toUpperCase(word.charAt(0));
                char last = Character.toUpperCase(word.charAt(len - 1));
                String middle = word.substring(1, len - 1);

                result += first + middle + last + " ";
            }
        }

        System.out.println(result.trim());
    }
}
```

#### 🔹 Method 2: Without Using Split
```java
import java.util.*;

public class Main {
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);
        String str = sc.nextLine();

        String result = "";
        int i = 0;

        while(i < str.length()){
            if(str.charAt(i) == ' '){
                result += " ";
                i++;
                continue;
            }

            int start = i;
            while(i < str.length() && str.charAt(i) != ' '){
                i++;
            }
            int end = i - 1;

            if(start == end){
                result += Character.toUpperCase(str.charAt(start));
            } else {
                result += Character.toUpperCase(str.charAt(start));

                for(int j = start + 1; j < end; j++){
                    result += str.charAt(j);
                }

                result += Character.toUpperCase(str.charAt(end));
            }
        }

        System.out.println(result);
    }
}
```
