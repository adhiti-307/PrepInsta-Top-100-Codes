# 18. Replace Substring in a String

## 📌 Problem Statement

Write a program to replace all occurrences of a given substring in a string with another string.

---

## 📝 Input Format

* First line: A string `str`
* Second line: Substring to be replaced `substr`
* Third line: Replacement string `replace`

---

## 📤 Output Format

* Print the updated string after replacing all occurrences

---

## 🔍 Constraints

* Strings may contain alphabets, digits, and special characters
* Replacement should be done for **all occurrences**

---

## 💡 Approach (Explanation)

1. Read the input strings: original string, substring, and replacement
2. If the substring and replacement are the same, return the original string
3. Use the built-in `replace()` method to replace all occurrences of the substring
4. Print the modified string

---

## 💻 Java Code

```java id="3h5k2s"
import java.util.*;

class Main{
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);
        
        String str = sc.nextLine();
        String substr = sc.nextLine();
        String replace = sc.nextLine();

        if(substr.equals(replace)){
            System.out.println(str);
            return;
        }

        str = str.replace(substr, replace);

        System.out.println(str);
    }
}
```

---

## 🧪 Sample Input

```
hello world world
world
java
```

---

## ✅ Sample Output

```
hello java java
```

---

## 📊 Complexity Analysis

* Time Complexity: **O(n)**
* Space Complexity: **O(1)** (excluding input/output)
