## 🧩 Problem: Remove Brackets from an Algebraic Expression

### 📄 Problem Statement
Write a Java program to remove all brackets from a given algebraic expression.

### 📥 Input
A string `expr` representing an algebraic expression.

### 📤 Output
Print the expression after removing all brackets.

### 💡 Example

**Input:**
```
(a+b)
```

**Output:**
```
a+b
```

**Input:**
```
((x+y)*(z))
```

**Output:**
```
x+y*z
```

### 🧠 Approach
1. Traverse each character of the string.  
2. Ignore bracket characters `(` and `)`.  
3. Append all other characters to the result.  
4. Print the final expression.  

---

### 💻 Java Code

#### 🔹 Method 1: Using Loop
```java
import java.util.*;

public class Main {
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);
        String expr = sc.nextLine();

        String result = "";

        for(char ch : expr.toCharArray()){
            if(ch != '(' && ch != ')'){
                result += ch;
            }
        }

        System.out.println(result);
    }
}
```

---

#### 🔹 Method 2: Using Inbuilt Method
```java
import java.util.*;

public class Main {
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);
        String expr = sc.nextLine();

        String result = expr.replace("(", "").replace(")", "");

        System.out.println(result);
    }
}
```
