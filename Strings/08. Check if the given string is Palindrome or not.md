## 🧩 Problem: Check Whether a String is a Palindrome

### 📄 Problem Statement
Write a Java program to check whether a given string is a palindrome.

### 📥 Input
A string `str`.

### 📤 Output
Print `"Palindrome"` if the string reads the same forward and backward, otherwise print `"Not Palindrome"`.

### 💡 Example

**Input:**
```
madam
```

**Output:**
```
Palindrome
```

**Input:**
```
hello
```

**Output:**
```
Not Palindrome
```

### 🧠 Approach
1. Compare the string with its reverse.  
2. If both are equal → palindrome.  
3. Otherwise → not a palindrome.  

---

### 💻 Java Code

#### 🔹 Method 1: Using StringBuilder
```java
import java.util.*;

public class Main {
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);
        String str = sc.next();

        String reversed = new StringBuilder(str).reverse().toString();

        if(str.equals(reversed))
            System.out.println("Palindrome");
        else
            System.out.println("Not Palindrome");
    }
}
```

#### 🔹 Method 2: Without Built-in (Two Pointer)
```java
import java.util.*;

public class Main {
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);
        String str = sc.next();

        int left = 0, right = str.length() - 1;
        boolean isPalindrome = true;

        while(left < right){
            if(str.charAt(left) != str.charAt(right)){
                isPalindrome = false;
                break;
            }
            left++;
            right--;
        }

        if(isPalindrome)
            System.out.println("Palindrome");
        else
            System.out.println("Not Palindrome");
    }
}
```
