## 🧩 Problem: Check Whether a Character is a Vowel or Consonant

### 📄 Problem Statement
Write a Java program to check whether a given character is a vowel or a consonant.

### 📥 Input
A single character `ch`.

### 📤 Output
Print `"Vowel"` if the character is a vowel, otherwise print `"Consonant"`.

### 💡 Example

**Input:**
```
a
```

**Output:**
```
Vowel
```

**Input:**
```
z
```

**Output:**
```
Consonant
```

### 🧠 Approach
1. Convert the character to lowercase for uniform comparison.  
2. Check if it is one of: `a, e, i, o, u`.  
3. If yes → vowel, else → consonant.  

---

### 💻 Java Code
```java
import java.util.*;

public class Main {
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);
        char ch = sc.next().charAt(0);

        ch = Character.toLowerCase(ch);

        if(ch == 'a' || ch == 'e' || ch == 'i' || ch == 'o' || ch == 'u')
            System.out.println("Vowel");
        else
            System.out.println("Consonant");
    }
}
```
