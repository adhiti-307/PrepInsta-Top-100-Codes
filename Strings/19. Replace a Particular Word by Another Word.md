# 19. Replace a Particular Word by Another Word

## 📌 Problem Statement

Write a program to replace all occurrences of a given word in a sentence with another word such that the final output does not contain unnecessary extra spaces.

---

## 📝 Input Format

* First line: A sentence `str`
* Second line: Word to be replaced `oldWord`
* Third line: New word `newWord`

---

## 📤 Output Format

* Print the updated sentence after replacing all occurrences of the given word
* Ensure the output has **proper spacing (no extra spaces)**

---

## 🔍 Constraints

* The sentence may contain alphabets, digits, and spaces
* Replacement should be done for **all occurrences of the whole word**

---

## 💡 Approach (Explanation)

1. Read the input sentence and the two words
2. If both words are the same, return the original sentence
3. Use `replaceAll()` with word boundary (`\\b`) to replace only whole words
4. Normalize spaces using `replaceAll("\\s+", " ")`
5. Trim leading and trailing spaces using `trim()`
6. Print the final result

---

## 💻 Java Code

```java id="f92k1x"
import java.util.*;

class Main{
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);

        String str = sc.nextLine();
        String oldWord = sc.nextLine();
        String newWord = sc.nextLine();

        if(oldWord.equals(newWord)){
            System.out.println(str.trim());
            return;
        }

        // Replace whole word occurrences
        str = str.replaceAll("\\b" + oldWord + "\\b", newWord);

        // Normalize spaces and trim
        str = str.replaceAll("\\s+", " ").trim();

        System.out.println(str);
    }
}
```

## 📊 Complexity Analysis

* Time Complexity: **O(n)**
* Space Complexity: **O(1)** (excluding input/output)

---

## 🔥 Key Points

* `\\b` ensures only **whole words** are replaced
* `\\s+` removes multiple spaces
* `trim()` removes leading and trailing spaces
