## 🧩 Problem: Calculate Frequency of Characters in a String

### 📄 Problem Statement
Write a Java program to count the frequency of each character in a given string.

### 📥 Input
A string `str`.

### 📤 Output
Print each character along with its frequency.

### 💡 Example

**Input:**
```
hello
```

**Output:**
```
h : 1
e : 1
l : 2
o : 1
```

---

### 🧠 Approach
1. Traverse the string character by character.  
2. For each character, count how many times it appears.  
3. Mark characters as visited to avoid recounting.  
4. Print character and its count.  

---

### 💻 Java Code

#### 🔹 Method 1: Using Visited Array (No Extra DS)
```java
import java.util.*;

public class Main {
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);
        String str = sc.nextLine();

        boolean[] visited = new boolean[str.length()];

        for(int i = 0; i < str.length(); i++){
            if(visited[i]) continue;

            int count = 1;

            for(int j = i + 1; j < str.length(); j++){
                if(str.charAt(i) == str.charAt(j)){
                    visited[j] = true;
                    count++;
                }
            }

            System.out.println(str.charAt(i) + " : " + count);
        }
    }
}
```

#### 🔹 Method 2: Using HashMap (Optimized)
```java
import java.util.*;

public class Main {
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);
        String str = sc.nextLine();

        HashMap<Character, Integer> map = new HashMap<>();

        for(char ch : str.toCharArray()){
            map.put(ch, map.getOrDefault(ch, 0) + 1);
        }

        for(char key : map.keySet()){
            System.out.println(key + " : " + map.get(key));
        }
    }
}
```
