## 🧩 Problem: Print the Given String in Reverse Order

### 📄 Problem Statement
Write a Java program to print the given string in reverse order.

### 📥 Input
A string `str`.

### 📤 Output
Print the reversed string.

### 💡 Example

**Input:**
```
hello
```

**Output:**
```
olleh
```

**Input:**
```
JAVA
```

**Output:**
```
AVAJ
```

### 🧠 Approach
1. Traverse the string from the last character to the first.  
2. Append each character to a result string.  
3. Print the reversed string.  

---

### 💻 Java Code

#### 🔹 Method 1: Using StringBuilder
```java
import java.util.*;

public class Main {
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);
        String str = sc.next();

        String reversed = new StringBuilder(str).reverse().toString();

        System.out.println(reversed);
    }
}
```

---

#### 🔹 Method 2: Without Built-in Method
```java
import java.util.*;

public class Main {
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);
        String str = sc.next();

        String result = "";

        for(int i = str.length() - 1; i >= 0; i--){
            result += str.charAt(i);
        }

        System.out.println(result);
    }
}
```
