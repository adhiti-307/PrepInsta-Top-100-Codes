# 21. Count Common Subsequences in Two Strings

## 📌 Problem Statement

Write a program to count the number of **common subsequences** between two given strings.

A **subsequence** is a sequence that can be derived from another string by deleting some or no characters without changing the order of the remaining characters.

---

## 📝 Input Format

* First line: String `str1`
* Second line: String `str2`

---

## 📤 Output Format

* Print the total number of common subsequences

---

## 🔍 Constraints

* Strings may contain lowercase/uppercase characters
* Length of strings can be up to reasonable limits (as per constraints)

---

## 💡 Approach (Explanation)

1. Let `dp[i][j]` represent the number of common subsequences between:

   * `str1[0..i-1]` and `str2[0..j-1]`

2. Initialize a 2D array `dp` of size `(n+1) × (m+1)` with `0`

3. Traverse both strings:

   * If characters match:

     ```
     dp[i][j] = 1 + dp[i-1][j] + dp[i][j-1]
     ```
   * If characters do not match:

     ```
     dp[i][j] = dp[i-1][j] + dp[i][j-1] - dp[i-1][j-1]
     ```

4. The final answer will be stored in `dp[n][m]`

---

## 💻 Java Code

```java id="count-common-subseq"
import java.util.*;

class Main {

    static int countCommonSubsequences(String s, String t) {
        int n = s.length();
        int m = t.length();

        int[][] dp = new int[n + 1][m + 1];

        for (int i = 1; i <= n; i++) {
            for (int j = 1; j <= m; j++) {

                if (s.charAt(i - 1) == t.charAt(j - 1)) {
                    dp[i][j] = 1 + dp[i - 1][j] + dp[i][j - 1];
                } else {
                    dp[i][j] = dp[i - 1][j] + dp[i][j - 1] - dp[i - 1][j - 1];
                }

            }
        }

        return dp[n][m];
    }

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        String str1 = sc.nextLine();
        String str2 = sc.nextLine();

        System.out.println(countCommonSubsequences(str1, str2));
    }
}
```

## 📊 Complexity Analysis

* Time Complexity: **O(n × m)**
* Space Complexity: **O(n × m)**

---

## 🔥 Key Points

* This problem counts **all common subsequences**, not just the longest
* Uses **Dynamic Programming (DP)**
* Subtraction in the formula avoids **double counting**
* Different from **Longest Common Subsequence (LCS)**
