## 🧩 Problem: Nth Term of Fibonacci Series

### 📄 Problem Statement
Write a Java program to find the `n`th term of the Fibonacci series.

### 📥 Input
A single integer `n`.

### 📤 Output
Print the `n`th Fibonacci number.

### 💡 Example

**Input:**
```
5
```

**Output:**
```
3
```

**Input:**
```
7
```

**Output:**
```
8
```

### 🧠 Approach
1. **Using Iteration:**  
   Start with `0` and `1`, iterate up to `n` and keep updating values.

2. **Using Recursion:**  
   Use the recursive definition:  
   `F(n) = F(n-1) + F(n-2)` with base cases.

3. **Using Static Variable (Memoization style):**  
   Store previously computed values to avoid recomputation.

4. **Using Direct Formula (Binet’s Formula):**  
   Use the mathematical formula involving powers and square roots.

---

### 💻 Java Code

#### 🔹 Method 1: Using Iteration
```java
import java.util.*;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int n = sc.nextInt();

        int a = 0, b = 1;

        if(n == 1) {
            System.out.println(a);
            return;
        }

        for(int i = 2; i <= n; i++) {
            int next = a + b;
            a = b;
            b = next;
        }

        System.out.println(a);
    }
}
```

#### 🔹 Method 2: Using Recursion
```java
import java.util.*;

public class Main {

    public static int fib(int n) {
        if(n == 1) return 0;
        if(n == 2) return 1;
        return fib(n-1) + fib(n-2);
    }

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int n = sc.nextInt();
        System.out.println(fib(n));
    }
}
```

#### 🔹 Method 3: Using Static Variable (Memoization)
```java
import java.util.*;

public class Main {

    static int[] dp = new int[1000];

    public static int fib(int n) {
        if(n == 1) return 0;
        if(n == 2) return 1;

        if(dp[n] != 0) return dp[n];

        return dp[n] = fib(n-1) + fib(n-2);
    }

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int n = sc.nextInt();
        System.out.println(fib(n));
    }
}
```

#### 🔹 Method 4: Using Direct Formula (Binet’s Formula)
```java
import java.util.*;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int n = sc.nextInt();

        double phi = (1 + Math.sqrt(5)) / 2;
        double result = Math.pow(phi, n - 1) / Math.sqrt(5);

        System.out.println((int)Math.round(result));
    }
}
```
