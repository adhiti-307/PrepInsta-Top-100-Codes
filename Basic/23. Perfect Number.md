## 🧩 Problem: Check Whether a Number is a Perfect Number

### 📄 Problem Statement
Write a Java program to check whether a number is perfect. A perfect number is a positive integer that equals the sum of its proper positive divisors (excluding the number itself).

### 📥 Input
A single integer `n`.

### 📤 Output
Print `"Perfect Number"` or `"Not Perfect Number"`.

### 💡 Example

**Input:**
```
6
```

**Output:**
```
Perfect Number
```

**Explanation:**
```
Factors of 6 (except 6 itself) = {1, 2, 3}
1 + 2 + 3 = 6
```

### 🧠 Approach
1. Find all divisors except the number itself.  
2. Sum them.  
3. Compare with original number.  

---

### 💻 Java Code
```java
import java.util.*;

public class Main {
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);
        int n = sc.nextInt();
        int sum = 0;

        for(int i = 1; i <= n/2; i++){
            if(n % i == 0) sum += i;
        }

        System.out.println(sum == n ? "Perfect Number" : "Not Perfect Number");
    }
}
```
