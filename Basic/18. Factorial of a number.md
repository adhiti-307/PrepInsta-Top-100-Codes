## 🧩 Problem: Factorial of a Number

### 📄 Problem Statement
Write a Java program to find the factorial of a given number.

### 📥 Input
A single integer `n`.

### 📤 Output
Print the factorial of `n`.

### 💡 Example

**Input:**
```
5
```

**Output:**
```
120
```

**Input:**
```
3
```

**Output:**
```
6
```

### 🧠 Approach
1. **Using Iteration:**  
   Multiply numbers from `1` to `n` using a loop.

2. **Using Recursion:**  
   Use the relation `n! = n × (n-1)!` with base case `0! = 1`.

3. **Using Dynamic Programming (Memoization):**  
   Store previously computed factorial values to avoid recomputation.

4. **Using BigInteger (for large values):**  
   Use `BigInteger` to handle very large factorials beyond integer limits.

---

### 💻 Java Code

#### 🔹 Method 1: Using Iteration
```java
import java.util.*;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int n = sc.nextInt();

        long fact = 1;

        for(int i = 1; i <= n; i++) {
            fact *= i;
        }

        System.out.println(fact);
    }
}
```

---

#### 🔹 Method 2: Using Recursion
```java
import java.util.*;

public class Main {

    public static long factorial(int n) {
        if(n == 0 || n == 1) return 1;
        return n * factorial(n - 1);
    }

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int n = sc.nextInt();
        System.out.println(factorial(n));
    }
}
```

---

#### 🔹 Method 3: Using Memoization
```java
import java.util.*;

public class Main {

    static long[] dp = new long[1000];

    public static long factorial(int n) {
        if(n == 0 || n == 1) return 1;

        if(dp[n] != 0) return dp[n];

        return dp[n] = n * factorial(n - 1);
    }

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int n = sc.nextInt();
        System.out.println(factorial(n));
    }
}
```

---

#### 🔹 Method 4: Using BigInteger (for large n)
```java
import java.util.*;
import java.math.BigInteger;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int n = sc.nextInt();

        BigInteger fact = BigInteger.ONE;

        for(int i = 1; i <= n; i++) {
            fact = fact.multiply(BigInteger.valueOf(i));
        }

        System.out.println(fact);
    }
}
```
