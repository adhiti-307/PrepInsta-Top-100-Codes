## 🧩 Problem: Check Whether a Number is a Strong Number

### 📄 Problem Statement
Write a Java program to check whether a given number is a Strong number. A strong number (or Krishnamurthy number) is a special number where the sum of the factorial of its digits equals the original number.

### 📥 Input
A single integer `n`.

### 📤 Output
Print `"Strong Number"` or `"Not Strong Number"`.

### 💡 Example

**Input:**
```
145
```

**Output:**
```
Strong Number
```

**Explanation:**
```
1! + 4! + 5! = 1 + 24 + 120 = 145
```

### 🧠 Approach
1. Extract each digit.  
2. Find factorial of each digit.  
3. Sum all factorials.  
4. Compare with original number.  

---

### 💻 Java Code
```java
import java.util.*;

public class Main {
    static int factorial(int n){
        int fact = 1;
        for(int i = 1; i <= n; i++) fact *= i;
        return fact;
    }

    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);
        int n = sc.nextInt();
        int temp = n, sum = 0;

        while(temp > 0){
            int digit = temp % 10;
            sum += factorial(digit);
            temp /= 10;
        }

        System.out.println(sum == n ? "Strong Number" : "Not Strong Number");
    }
}
```
