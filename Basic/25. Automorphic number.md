## 🧩 Problem: Check Whether a Number is an Automorphic Number

### 📄 Problem Statement
Write a Java program to check whether a number is automorphic.An automorphic number (or circular number) is a natural number whose square ends in the same digits as the number itself

### 📥 Input
A single integer `n`.

### 📤 Output
Print `"Automorphic"` or `"Not Automorphic"`.

### 💡 Example

**Input:**
```
76
```

**Output:**
```
Automorphic
```

**Explanation:**
```
76 x 76 = 17**76**
```

### 🧠 Approach
1. Square the number.  
2. Check if square ends with the number.  

---

### 💻 Java Code
```java
import java.util.*;

public class Main {
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);
        int n = sc.nextInt();

        int square = n * n;

        System.out.println(String.valueOf(square).endsWith(String.valueOf(n)) 
            ? "Automorphic" : "Not Automorphic");
    }
}
```
