## 🧩 Problem: Check Whether Two Numbers are a Friendly Pair

### 📄 Problem Statement
Write a Java program to check whether two numbers are friendly pairs.A friendly pair (or friendly numbers) are two natural numbers that share the same abundancy index, meaning the sum of their proper divisors divided by the number itself is equal.

### 📥 Input
Two integers `a` and `b`.

### 📤 Output
Print `"Friendly Pair"` or `"Not Friendly Pair"`.

### 💡 Example

**Input:**
```
6 28
```

**Output:**
```
Friendly Pair
```

**Explanation**
 - For **6**: Divisors are 1, 2, 3, 6.<br>
Sum = 1 + 2 + 3 + 6 = 12.<br>
Ratio = 12 / 6 = 2.
 - For **28**: Divisors are 1, 2, 4, 7, 14, 28.<br>
Sum = 1 + 2 + 4 + 7 + 14 + 28 = 56.<br>
Ratio = 56 / 28 = 2.

### 🧠 Approach
1. Find sum of divisors for both numbers.  
2. Divide by respective numbers.  
3. Compare ratios.  

---

### 💻 Java Code
```java
import java.util.*;

public class Main {

    public static int sumDiv(int n){
        int sum = 0;
        for(int i = 1; i <= n; i++){
            if(n % i == 0) sum += i;
        }
        return sum;
    }

    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);
        int a = sc.nextInt();
        int b = sc.nextInt();

        int sum1 = sumDiv(a);
        int sum2 = sumDiv(b);

        if(sum1 * b == sum2 * a)
            System.out.println("Friendly Pair");
        else
            System.out.println("Not Friendly Pair");
    }
}
```
