## 🧩 Problem: Sum of Numbers in a Given Range

### 📄 Problem Statement
Write a Java program to calculate the sum of all integers within a given range.

### 📥 Input
The first line contains an integer `start`.  
The second line contains an integer `end`.

### 📤 Output
Print a single integer representing the sum of all numbers from `start` to `end` (inclusive).

### 💡 Example

**Input:**
```
1
5
```

**Output:**
```
15
```

**Input:**
```
3
7
```

**Output:**
```
25
```

### 🧠 Approach
Iterate from `start` to `end` and keep adding each number to a sum variable.

---

### 💻 Java Code
```java
import java.util.*;

public class Main{
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);
        int n = sc.nextInt();
        sc.nextLine();
        int m = sc.nextInt();

        int sum = 0;
        for(int i = n; i <= m; i++){
            sum += i;
        }
        System.out.println(sum);
        
    }
}
```
