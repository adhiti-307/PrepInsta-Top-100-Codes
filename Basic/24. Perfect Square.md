## 🧩 Problem: Check Whether a Number is a Perfect Square

### 📄 Problem Statement
Write a Java program to check if a number is a perfect square.

### 📥 Input
A single integer `n`.

### 📤 Output
Print `"Perfect Square"` or `"Not Perfect Square"`.

### 💡 Example

**Input:**
```
25
```

**Output:**
```
Perfect Square
```

### 🧠 Approach
1. Compute square root of number.  
2. Check if square of root equals original number.  

---

### 💻 Java Code
```java
import java.util.*;

public class Main {
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);
        int n = sc.nextInt();

        int root = (int)Math.sqrt(n);

        System.out.println(root * root == n ? "Perfect Square" : "Not Perfect Square");
    }
}
```
