## 🧩 Problem: Reverse of a Number

### 📄 Problem Statement
Write a Java program to reverse the digits of a given integer.

### 📥 Input
A single integer `n`.

### 📤 Output
Print the reversed number.

### 💡 Example

**Input:**
```
1234
```

**Output:**
```
4321
```

**Input:**
```
907
```

**Output:**
```
709
```
### 🧠 Approach
1. Extract each digit using modulus (`% 10`) and build the reversed number by multiplying the current result by 10 and adding the digit. Continue until the number becomes 0.  

2. Convert the number to a string, reverse the string using built-in methods (like `StringBuilder`), and then convert it back to an integer.  

3. Traverse the string from end to start and build the reversed result by appending each character.


### 💻 Java Code Approach 1
```java
import java.util.*;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int n = sc.nextInt();

        int rev_n = 0;
        while(n > 0){
            int r = n % 10;
            rev_n = rev_n * 10 + r;
            n /= 10;
        }
        System.out.println(rev_n);
    }
}
```
### 💻 Java Code Approach 2
```java
import java.util.*;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int n = sc.nextInt();

        StringBuilder sb = new StringBuilder(String.valueOf(n));
        sb.reverse();

        int reversed = Integer.parseInt((sb.toString()));
        System.out.println(reversed);
    }
}
```
### 💻 Java Code Approach 3
```java
import java.util.*;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        String str = sc.nextLine();

        String rev = "";
        for(int i = str.length(); i > 0; i--){
            rev += i;
        }
        System.out.println(rev);
    }
}
```
