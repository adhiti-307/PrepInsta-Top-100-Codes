## 🧩 Problem: Power of a Number

### 📄 Problem Statement
Write a Java program to calculate the power of a number, i.e., compute `base^exponent`.

### 📥 Input
The first line contains an integer `base`.  
The second line contains an integer `exponent`.

### 📤 Output
Print the value of `base` raised to the power `exponent`.

### 💡 Example

**Input:**
```
2
3
```

**Output:**
```
8
```

**Input:**
```
5
4
```

**Output:**
```
625
```

### 🧠 Approach
1. **Using Iteration:**  
   Multiply the base repeatedly `exponent` times.

2. **Using Recursion:**  
   Use the relation `power(b, e) = b × power(b, e-1)` with base case `e = 0`.

3. **Using Fast Exponentiation (Binary Exponentiation):**  
   Reduce the exponent by half each time for better efficiency.

4. **Using Built-in Function:**  
   Use `Math.pow(base, exponent)`.

---

### 💻 Java Code

#### 🔹 Method 1: Using Iteration
```java
import java.util.*;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int base = sc.nextInt();
        int exp = sc.nextInt();

        long result = 1;

        for(int i = 1; i <= exp; i++) {
            result *= base;
        }

        System.out.println(result);
    }
}
```

---

#### 🔹 Method 2: Using Recursion
```java
import java.util.*;

public class Main {

    public static long power(int base, int exp) {
        if(exp == 0) return 1;
        return base * power(base, exp - 1);
    }

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int base = sc.nextInt();
        int exp = sc.nextInt();

        System.out.println(power(base, exp));
    }
}
```

---

#### 🔹 Method 3: Fast Exponentiation (Optimal)
```java
import java.util.*;

public class Main {

    public static long power(int base, int exp) {
        if(exp == 0) return 1;

        long half = power(base, exp / 2);

        if(exp % 2 == 0)
            return half * half;
        else
            return base * half * half;
    }

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int base = sc.nextInt();
        int exp = sc.nextInt();

        System.out.println(power(base, exp));
    }
}
```

---

#### 🔹 Method 4: Using Built-in Function
```java
import java.util.*;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int base = sc.nextInt();
        int exp = sc.nextInt();

        System.out.println((long)Math.pow(base, exp));
    }
}
```
