## 🧩 Problem: Check Whether a Number is a Palindrome

### 📄 Problem Statement
Write a Java program to check whether a given integer is a palindrome number.

### 📥 Input
A single integer `n`.

### 📤 Output
Print `"Palindrome"` if the number reads the same forward and backward, otherwise print `"Not Palindrome"`.

### 💡 Example

**Input:**
```
121
```

**Output:**
```
Palindrome
```

**Input:**
```
123
```

**Output:**
```
Not Palindrome
```

### 🧠 Approach
1. Reverse the number using modulus (`% 10`) and division (`/ 10`), then compare it with the original number.  

2. Convert the number to a string and check if it is equal to its reverse.  

3. Traverse the string from both ends (two-pointer approach) and compare characters.

---

### 💻 Java Code Approach 1
```java
import java.util.*;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int n = sc.nextInt();
        int original = n, rev = 0;

        while(n > 0) {
            int digit = n % 10;
            rev = rev * 10 + digit;
            n /= 10;
        }

        if(original == rev)
            System.out.println("Palindrome");
        else
            System.out.println("Not Palindrome");
    }
}
```
### 💻 Java Code Approach 2
```java
import java.util.*;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        String str = sc.next();

        String reversed = new StringBuilder(str).reverse().toString();

        if(str.equals(reversed))
            System.out.println("Palindrome");
        else
            System.out.println("Not Palindrome");
    }
}
```
### 💻 Java Code Approach 3
```java
import java.util.*;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        String str = sc.next();

        int left = 0, right = str.length() - 1;
        boolean isPalindrome = true;

        while(left < right) {
            if(str.charAt(left) != str.charAt(right)) {
                isPalindrome = false;
                break;
            }
            left++;
            right--;
        }

        if(isPalindrome)
            System.out.println("Palindrome");
        else
            System.out.println("Not Palindrome");
    }
}
```
