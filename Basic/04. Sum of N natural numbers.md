## 🧩 Problem: Sum of First N Natural Numbers (Using Formula)

### 📄 Problem Statement

Write a Java program to calculate the sum of the first `n` natural numbers using the mathematical formula.

### 📥 Input

An integer `n`.

### 📤 Output

Print the sum of the first `n` natural numbers.

### 💡 Example

- Input: `5`<br>
  Output: `15`

- Input: `10`<br>
  Output: `55`

### 🧠 Approach

Use the formula:
Sum = n × (n + 1) / 2

---

### 💻 Java Code

```java
import java.util.*;

public class Main{
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);
        int n = sc.nextInt();

        int sum = n * (n + 1) / 2;
        System.out.println(sum);
        
    }
}
```
