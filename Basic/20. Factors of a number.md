## 🧩 Problem: Find All Factors of a Number

### 📄 Problem Statement
Write a Java program to find all factors of a given number.

### 📥 Input
A single integer `n`.

### 📤 Output
Print all factors of `n` in ascending order.

### 💡 Example

**Input:**
```
12
```

**Output:**
```
1 2 3 4 6 12
```

**Input:**
```
7
```

**Output:**
```
1 7
```

### 🧠 Approach

1. **Using Range [2, number]**  
   Check all numbers from `1` to `n` and print those that divide `n`.

2. **Using Range [2, number/2]**  
   Factors (except `n`) cannot be greater than `n/2`.

3. **Using Range [2, √number]**  
   Factors come in pairs → check till square root.

4. **Using Range [2, √number] (Optimized Output)**  
   Print both divisor and quotient together for efficiency.

5. **Handling Negative Numbers**  
   Convert negative input to positive using absolute value.

---

### 💻 Java Code

#### 🔹 Method 1: Full Range
```java
import java.util.*;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int n = sc.nextInt();

        for(int i = 1; i <= n; i++) {
            if(n % i == 0)
                System.out.print(i + " ");
        }
    }
}
```

#### 🔹 Method 2: Up to n/2
```java
import java.util.*;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int n = sc.nextInt();

        for(int i = 1; i <= n/2; i++) {
            if(n % i == 0)
                System.out.print(i + " ");
        }
        System.out.print(n);
    }
}
```

---

#### 🔹 Method 3: Up to √n
```java
import java.util.*;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int n = sc.nextInt();

        for(int i = 1; i * i <= n; i++) {
            if(n % i == 0) {
                System.out.print(i + " ");
            }
        }
    }
}
```

---

#### 🔹 Method 4: Optimized (Pair Factors)
```java
import java.util.*;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int n = sc.nextInt();

        ArrayList<Integer> list = new ArrayList<>();

        for(int i = 1; i * i <= n; i++) {
            if(n % i == 0) {
                System.out.print(i + " ");
                if(i != n / i)
                    list.add(n / i);
            }
        }

        for(int i = list.size() - 1; i >= 0; i--) {
            System.out.print(list.get(i) + " ");
        }
    }
}
```

#### 🔹 Method 5: Handling Negative Numbers
```java
import java.util.*;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int n = sc.nextInt();

        n = Math.abs(n);

        for(int i = 1; i * i <= n; i++) {
            if(n % i == 0) {
                System.out.print(i + " ");
                if(i != n / i)
                    System.out.print(n / i + " ");
            }
        }
    }
}
```
