## 🧩 Problem: Find the Greatest of Three Numbers

### 📄 Problem Statement
Write a Java program to find the greatest among three given integers.

### 📥 Input
A single line containing three integers `a`, `b`, and `c` separated by space.

### 📤 Output
Print the greatest number among `a`, `b`, and `c`.

### 💡 Example

**Input:**
```
10 25 15
```

**Output:**
```
25
```

**Input:**
```
7 3 9
```

**Output:**
```
9
```

### 🧠 Approach
Use nested ternary operators to compare the three numbers and determine the largest value.

---

### 💻 Java Code
```java
import java.util.*;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        
        int a = sc.nextInt();
        int b = sc.nextInt();
        int c = sc.nextInt();
        
        int max = (a > b) ? (a > c ? a : c) : (b > c ? b : c);
        
        System.out.println(max);
    }
}
```
