## 🧩 Problem: Find Prime Numbers Within a Given Range

### 📄 Problem Statement
Write a Java program to print all prime numbers within a given range.

### 📥 Input
The first line contains an integer `start`.  
The second line contains an integer `end`.

### 📤 Output
Print all prime numbers between `start` and `end` (inclusive), separated by space.

### 💡 Example

**Input:**
```
1
10
```

**Output:**
```
2 3 5 7
```

**Input:**
```
10
20
```

**Output:**
```
11 13 17 19
```

### 🧠 Approach (Most Optimal)
Use the **Sieve of Eratosthenes** algorithm:
- Create a boolean array to mark prime numbers  
- Initialize all numbers as prime  
- Start marking multiples of each prime from `2` onwards  
- Collect all unmarked numbers within the range  

Time Complexity: **O(n log log n)**

---

### 💻 Java Code
```java
import java.util.*;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        int start = sc.nextInt();
        int end = sc.nextInt();

        boolean[] isPrime = new boolean[end + 1];
        Arrays.fill(isPrime, true);

        if (end >= 0) isPrime[0] = false;
        if (end >= 1) isPrime[1] = false;

        for (int i = 2; i * i <= end; i++) {
            if (isPrime[i]) {
                for (int j = i * i; j <= end; j += i) {
                    isPrime[j] = false;
                }
            }
        }

        for (int i = Math.max(2, start); i <= end; i++) {
            if (isPrime[i]) {
                System.out.print(i + " ");
            }
        }
    }
}
```
