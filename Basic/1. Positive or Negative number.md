## 🧩 Problem: Check Whether a Number is Positive, Negative, or Zero

### 📄 Problem Statement

Write a Java program to check whether a given integer is positive, negative, or zero.

### 📥 Input

An integer `n`

### 📤 Output

Print:

* "Positive"
* "Negative"
* "Zero" 

### 💡 Example

Input: `5`
Output: `Positive`

---

### 💻 Java Code

```java
import java.util.*;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int n = sc.nextInt();

        if(n > 0) {
            System.out.println("Positive");
        } else if(n < 0) {
            System.out.println("Negative");
        } else {
            System.out.println("Zero");
        }
    }
}
```
