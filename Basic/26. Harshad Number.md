## 🧩 Problem: Check Whether a Number is a Harshad Number

### 📄 Problem Statement
Write a Java program to check whether a number is divisible by the sum of its digits.

### 📥 Input
A single integer `n`.

### 📤 Output
Print `"Harshad Number"` or `"Not Harshad Number"`.

### 💡 Example

**Input:**
```
18
```

**Output:**
```
Harshad Number
```

### 🧠 Approach
1. Find sum of digits.  
2. Check divisibility.  

---

### 💻 Java Code
```java
import java.util.*;

public class Main {
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);
        int n = sc.nextInt();
        int temp = n, sum = 0;

        while(temp > 0){
            sum += temp % 10;
            temp /= 10;
        }

        System.out.println(n % sum == 0 ? "Harshad Number" : "Not Harshad Number");
    }
}
```
