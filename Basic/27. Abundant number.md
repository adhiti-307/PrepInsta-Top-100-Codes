## 🧩 Problem: Check Whether a Number is an Abundant Number

### 📄 Problem Statement
Write a Java program to check whether a number is abundant.

### 📥 Input
A single integer `n`.

### 📤 Output
Print `"Abundant Number"` or `"Not Abundant Number"`.

### 💡 Example

**Input:**
```
12
```

**Output:**
```
Abundant Number
```

**Explanation**
```
divisors: 1, 2, 3, 4, 6;
sum: 16 > 12
```

### 🧠 Approach
1. Find sum of proper divisors.  
2. If sum > number → abundant.  

---

### 💻 Java Code
```java
import java.util.*;

public class Main {
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);
        int n = sc.nextInt();
        int sum = 0;

        for(int i = 1; i <= n/2; i++){
            if(n % i == 0) sum += i;
        }

        System.out.println(sum > n ? "Abundant Number" : "Not Abundant Number");
    }
}
```
