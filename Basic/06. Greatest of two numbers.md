## 🧩 Problem: Find the Greatest of Two Numbers

### 📄 Problem Statement
Write a Java program to find the greatest of two given integers.

### 📥 Input
The first line contains an integer `a`.  
The second line contains an integer `b`.

### 📤 Output
Print the greater number among `a` and `b`.

### 💡 Example

**Input:**
```
10
20
```

**Output:**
```
20
```

**Input:**
```
7
3
```

**Output:**
```
7
```

### 🧠 Approach
Use the ternary operator to compare the two numbers and determine the greater one in a single line.

---

### 💻 Java Code
```java
import java.util.*;

public class Main{
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);
        int a = sc.nextInt();
        sc.nextLine();
        int b = sc.nextInt();

        int result = a > b ? a : b;
        System.out.println(result);
        
    }
}
```
