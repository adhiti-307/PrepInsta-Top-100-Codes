## 🧩 Problem: Sum of Digits of a Number

### 📄 Problem Statement
Write a Java program to calculate the sum of digits of a given integer.

### 📥 Input
A single integer `n`.

### 📤 Output
Print the sum of all digits of the number.

### 💡 Example

**Input:**
```
1234
```

**Output:**
```
10
```

**Input:**
```
567
```

**Output:**
```
18
```

### 🧠 Approach
Extract each digit using modulus (`% 10`) and add it to a sum variable, then remove the last digit using division (`/ 10`) until the number becomes 0.

---

### 💻 Java Code
```java
import java.util.*;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int n = sc.nextInt();

        int sum = 0;
        while(n > 0){
            int r = n % 10;
            sum += r;
            n /= 10;
        }
        System.out.println(sum);
    }
}
```
