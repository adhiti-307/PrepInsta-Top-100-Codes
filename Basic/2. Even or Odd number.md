## 🧩 Problem: Check Whether a Number is Even or Odd

### 📄 Problem Statement

Write a Java program to check whether a given integer is even or odd.

### 📥 Input

An integer `n`.

### 📤 Output

Print:

* `"Even"` if the number is divisible by 2
* `"Odd"` if the number is not divisible by 2

### 💡 Example

Input: `4`
Output: `Even`

Input: `7`
Output: `Odd`

---

### 💻 Java Code

```java
import java.util.*;

class hello{
    
    public static void main(String[] args){

        Scanner sc = new Scanner(System.in);
        int num = sc.nextInt();

        if(num % 2 == 0){
            System.out.println("Even");
        }else{
            System.out.println("Odd");
        }
    }
}
```
