## 🧩 Problem: Check Whether a Year is a Leap Year or Not

### 📄 Problem Statement
Write a Java program to check whether a given year is a leap year or not.

### 📥 Input
A single integer `year`.

### 📤 Output
Print `"Leap Year"` if the given year is a leap year, otherwise print `"Not a Leap Year"`.

### 💡 Example

**Input:**
```
2020
```

**Output:**
```
Leap Year
```

**Input:**
```
1900
```

**Output:**
```
Not a Leap Year
```

### 🧠 Approach
A year is a leap year if:
- It is divisible by 4 and not divisible by 100  
- OR it is divisible by 400  

---

### 💻 Java Code
```java
import java.util.*;

public class Main{
    public static void main (String[]args){
        Scanner sc = new Scanner(System.in);
        int year = sc.nextInt();

        if (year % 400 == 0){
        System.out.println ("Leap Year");
        }else if (year % 4 == 0 && year % 100 != 0){
        System.out.println ("Leap Year");
        }else{
            System.out.println ("Not a Leap Year");
        }
   }
 }
```
