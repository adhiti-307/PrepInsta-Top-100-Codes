## 🧩 Problem: Armstrong Numbers in a Given Range

### 📄 Problem Statement
Write a Java program to print all Armstrong numbers within a given range.

### 📥 Input
The first line contains an integer `start`.  
The second line contains an integer `end`.

### 📤 Output
Print all Armstrong numbers between `start` and `end` (inclusive), separated by space.

### 💡 Example

**Input:**
```
1
500
```

**Output:**
```
1 2 3 4 5 6 7 8 9 153 370 371 407
```

### 🧠 Approach
1. Iterate through all numbers from `start` to `end`.  
2. Count the number of digits of each number using a loop.  
3. Extract each digit using modulus (`% 10`).  
4. Compute the power of each digit using a custom function (avoid `Math.pow`).  
5. Add the values and compare with the original number.  

---

### 💻 Java Code
```java
import java.util.*;

public class Main {

    public static int power(int base, int exp) {
        int result = 1;
        for(int i = 0; i < exp; i++) {
            result *= base;
        }
        return result;
    }

    public static boolean isArmstrong(int n) {
        int original = n;
        
        int digits = 0, temp = n;
        while(temp > 0) {
            digits++;
            temp /= 10;
        }

        int sum = 0;
        temp = n;

        while(temp > 0) {
            int digit = temp % 10;
            sum += power(digit, digits);
            temp /= 10;
        }

        return sum == original;
    }

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int start = sc.nextInt();
        int end = sc.nextInt();

        for(int i = start; i <= end; i++) {
            if(isArmstrong(i)) {
                System.out.print(i + " ");
            }
        }
    }
}
```
