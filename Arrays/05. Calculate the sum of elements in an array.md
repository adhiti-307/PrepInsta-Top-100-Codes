# 5. Calculate the Sum of Elements in an Array

## 📌 Problem Statement

Write a program to calculate the **sum of all elements** in a given array.

---

## 📝 Input Format

* A single line containing space-separated integers (array elements)

---

## 📤 Output Format

* Print the sum of all elements

---

## 💡 Approach (Explanation)

1. Read the input array
2. Initialize a variable `sum = 0`
3. Traverse the array
4. Add each element to `sum`
5. Print the final value of `sum`

---

## 💻 Java Code

```java id="sum-array"
import java.util.*;

class Main {
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);

        String[] input = sc.nextLine().split("\\s+");
        int[] arr = new int[input.length];

        for(int i = 0; i < input.length; i++){
            arr[i] = Integer.parseInt(input[i]);
        }

        int sum = 0;

        for(int num : arr){
            sum += num;
        }

        System.out.println(sum);
    }
}
```

---

## 🧪 Sample Input

```text id="input1"
3 7 2 9 4
```

---

## ✅ Sample Output

```text id="output1"
25
```

---

## 📊 Complexity Analysis

* Time Complexity: **O(n)**
* Space Complexity: **O(1)**

---

## 🔥 Key Points

* Only one traversal is required
* Works for both positive and negative numbers
* Simple accumulation pattern
