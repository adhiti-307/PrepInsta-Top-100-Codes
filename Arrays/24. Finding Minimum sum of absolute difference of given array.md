# 24. Find Minimum Sum of Absolute Difference of an Array

## 📌 Problem Statement

Write a program to find the **minimum sum of absolute differences** between adjacent elements of an array.

👉 You are allowed to **rearrange the array** in any order.

---

## 📝 Input Format

* A single line containing space-separated integers (array elements)

---

## 📤 Output Format

* Print the minimum sum of absolute differences

---

## 💡 Approach (Explanation)

### 🔑 Key Idea

To minimize absolute differences:

> Arrange the array in **sorted order (ascending)**

Because:

* Closest values together → smallest differences
* Avoid large jumps between elements

---

### Steps:

1. Read the input array
2. Sort the array in ascending order
3. Initialize `sum = 0`
4. Traverse from `i = 1` to `n-1`:

   ```text
   sum += |arr[i] - arr[i-1]|
   ```
5. Print `sum`

---

## 💻 Java Code

```java id="min-abs-diff"
import java.util.*;

class Main {
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);

        String[] input = sc.nextLine().split("\\s+");
        int[] arr = new int[input.length];

        for(int i = 0; i < input.length; i++){
            arr[i] = Integer.parseInt(input[i]);
        }

        Arrays.sort(arr);

        int sum = 0;

        for(int i = 1; i < arr.length; i++){
            sum += Math.abs(arr[i] - arr[i - 1]);
        }

        System.out.println(sum);
    }
}
```

---

## 🧪 Sample Input

```text id="input1"
3 1 4 2
```

---

## ✅ Sample Output

```text id="output1"
3
```

---

## 🧠 Explanation

Sorted array:

```text id="exp1"
[1, 2, 3, 4]
```

Sum:

```text id="exp2"
|2-1| + |3-2| + |4-3|
= 1 + 1 + 1 = 3
```

---

## 📊 Complexity Analysis

* Time Complexity: **O(n log n)** (sorting)
* Space Complexity: **O(1)**

---

## 🔥 Key Points

* Sorting minimizes adjacent differences
* Greedy approach
* Works because nearby values reduce absolute gaps
