# 9. Finding the Frequency of Elements in an Array

## 📌 Problem Statement

Write a program to find the **frequency of each element** in a given array.

---

## 📝 Input Format

* A single line containing space-separated integers (array elements)

---

## 📤 Output Format

* Print each element along with its frequency

---

## 💡 Approach (Explanation)

1. Read the input array
2. Use a `HashMap` to store frequency:

   * Key → element
   * Value → count
3. Traverse the array:

   * For each element, update its frequency using `getOrDefault()`
4. Traverse the map using `entrySet()` to print element and frequency

---

## 💻 Java Code

```java id="freq-array"
import java.util.*;

class Main {
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);

        String[] input = sc.nextLine().split("\\s+");
        int[] arr = new int[input.length];

        for(int i = 0; i < input.length; i++){
            arr[i] = Integer.parseInt(input[i]);
        }

        HashMap<Integer, Integer> map = new HashMap<>();

        for(int num : arr){
            map.put(num, map.getOrDefault(num, 0) + 1);
        }

        for(Map.Entry<Integer, Integer> entry : map.entrySet()){
            System.out.println(entry.getKey() + " -> " + entry.getValue());
        }
    }
}
```

---

## 🧪 Sample Input

```text id="input1"
1 2 2 3 1 4 2
```

---

## ✅ Sample Output

```text id="output1"
1 -> 2
2 -> 3
3 -> 1
4 -> 1
```

---

## 📊 Complexity Analysis

* Time Complexity: **O(n)**
* Space Complexity: **O(n)**

---

## 🔥 Key Points

* `HashMap` provides efficient frequency counting
* `getOrDefault()` simplifies update logic
* `entrySet()` allows direct access to key-value pairs
