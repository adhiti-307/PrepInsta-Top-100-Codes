# 31. Find the Kth Minimum and Kth Maximum Element of an Array

## 📌 Problem Statement

Write a program to find the **Kth minimum** and **Kth maximum** element in a given array.

---

## 📝 Input Format

* First line: Space-separated integers (array elements)
* Second line: Integer `k`

---

## 📤 Output Format

* Print the Kth minimum element
* Print the Kth maximum element

---

## 💡 Approach (Explanation)

### 🔑 Simple Idea (Sorting)

1. Sort the array in ascending order
2. Kth minimum → `arr[k-1]`
3. Kth maximum → `arr[n-k]`

---

## 💻 Java Code

```java id="kth-min-max"
import java.util.*;

class Main {
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);

        String[] input = sc.nextLine().split("\\s+");
        int k = sc.nextInt();

        int n = input.length;
        int[] arr = new int[n];

        for(int i = 0; i < n; i++){
            arr[i] = Integer.parseInt(input[i]);
        }

        Arrays.sort(arr);

        if(k <= 0 || k > n){
            System.out.println("Invalid K");
            return;
        }

        int kthMin = arr[k - 1];
        int kthMax = arr[n - k];

        System.out.println("Kth Min: " + kthMin);
        System.out.println("Kth Max: " + kthMax);
    }
}
```

---

## 🧪 Sample Input

```text id="input1"
7 10 4 3 20 15
3
```

---

## ✅ Sample Output

```text id="output1"
Kth Min: 7
Kth Max: 10
```

---

## 🧠 Explanation

Sorted array:

```text id="exp1"
[3, 4, 7, 10, 15, 20]
```

* 3rd min → `7`
* 3rd max → `10`

---

## 📊 Complexity Analysis

* Time Complexity: **O(n log n)**
* Space Complexity: **O(1)**

---

## 🔥 Key Points

* Sorting is simplest approach
* Indexing is important (`k-1`, `n-k`)
* Validate `k` before accessing
