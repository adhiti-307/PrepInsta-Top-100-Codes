# 12. Count Distinct Elements in an Array

## 📌 Problem Statement

Write a program to count the number of **distinct (unique) elements** in a given array.

---

## 📝 Input Format

* A single line containing space-separated integers (array elements)

---

## 📤 Output Format

* Print the count of distinct elements

---

## 💡 Approach (Explanation)

1. Read the input array
2. Use a `HashSet` to store unique elements
3. Traverse the array and add each element to the set
4. Since a set does not allow duplicates, it will store only distinct values
5. Print the size of the set

---

## 💻 Java Code

```java id="count-distinct"
import java.util.*;

class Main {
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);

        String[] input = sc.nextLine().split("\\s+");
        int[] arr = new int[input.length];

        for(int i = 0; i < input.length; i++){
            arr[i] = Integer.parseInt(input[i]);
        }

        HashSet<Integer> set = new HashSet<>();

        for(int num : arr){
            set.add(num);
        }

        System.out.println(set.size());
    }
}
```

---

## 🧪 Sample Input

```text id="input1"
1 2 2 3 1 4 2
```

---

## ✅ Sample Output

```text id="output1"
4
```

---

## 📊 Complexity Analysis

* Time Complexity: **O(n)**
* Space Complexity: **O(n)**

---

## 🔥 Key Points

* `HashSet` automatically removes duplicates
* Efficient way to count unique elements
* No need for sorting
