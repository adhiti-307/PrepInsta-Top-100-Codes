# 34. Find Largest Sum Contiguous Subarray (Kadane’s Algorithm)

## 📌 Problem Statement

Given an array of integers (positive and negative), find the **maximum sum of a contiguous subarray**.

---

## 📝 Input Format

* A single line containing space-separated integers (array elements)

---

## 📤 Output Format

* Print the maximum subarray sum

---

## 💡 Approach (Explanation)

### 🔑 Key Idea (Kadane’s Algorithm)

At each index, decide:

```text id="kad1"
Do we extend the previous subarray OR start a new one?
```

---

### 🔁 Logic

* Maintain two variables:

  ```text id="kad2"
  currentSum → sum of current subarray  
  maxSum     → best answer so far  
  ```

* Traverse the array:

  ```text
  currentSum = max(arr[i], currentSum + arr[i])
  maxSum = max(maxSum, currentSum)
  ```

---

## 💻 Java Code

```java id="kadane"
import java.util.*;

class Main {
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);

        String[] input = sc.nextLine().split("\\s+");
        int[] arr = new int[input.length];

        for(int i = 0; i < input.length; i++){
            arr[i] = Integer.parseInt(input[i]);
        }

        int currentSum = arr[0];
        int maxSum = arr[0];

        for(int i = 1; i < arr.length; i++){
            currentSum = Math.max(arr[i], currentSum + arr[i]);
            maxSum = Math.max(maxSum, currentSum);
        }

        System.out.println(maxSum);
    }
}
```

---

## 🧪 Sample Input

```text id="input1"
-2 -3 4 -1 -2 1 5 -3
```

---

## ✅ Sample Output

```text id="output1"
7
```

---

## 🧠 Explanation

Subarray:

```text id="exp1"
[4, -1, -2, 1, 5]
```

Sum:

```text id="exp2"
4 -1 -2 +1 +5 = 7
```

---

## 📊 Complexity Analysis

* Time Complexity: **O(n)**
* Space Complexity: **O(1)**

---

## 🔥 Key Points

* Works in a single pass
* Handles negative numbers
* Best known solution for this problem

---

## ⚠️ Edge Case

If all elements are negative:

```text id="edge1"
Return the largest (least negative) element
```

---

## ⚡ One-line memory trick

```text id="mem1"
Extend or restart → whichever gives better sum
```
