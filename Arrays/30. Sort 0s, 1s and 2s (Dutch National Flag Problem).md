# 32. Sort 0s, 1s and 2s (Dutch National Flag Problem)

## 📌 Problem Statement

Given an array consisting of only **0s, 1s, and 2s**, sort the array **without using any sorting algorithm**.

---

## 📝 Input Format

* A single line containing space-separated integers (only 0, 1, 2)

---

## 📤 Output Format

* Print the sorted array

---

## 💡 Approach (Explanation)

### 🔑 Key Idea (Dutch National Flag Algorithm)

We use **three pointers**:

```text
low  → for 0s  
mid  → current element  
high → for 2s  
```

---

### 🔁 Rules

* If `arr[mid] == 0`
  → swap with `low`, move both `low++` and `mid++`

* If `arr[mid] == 1`
  → just move `mid++`

* If `arr[mid] == 2`
  → swap with `high`, move `high--` (do NOT move mid)

---

## 💻 Java Code

```java id="dnf-sort"
import java.util.*;

class Main {
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);

        String[] input = sc.nextLine().split("\\s+");
        int[] arr = new int[input.length];

        for(int i = 0; i < input.length; i++){
            arr[i] = Integer.parseInt(input[i]);
        }

        int low = 0, mid = 0, high = arr.length - 1;

        while(mid <= high){
            if(arr[mid] == 0){
                int temp = arr[low];
                arr[low] = arr[mid];
                arr[mid] = temp;
                low++;
                mid++;
            }
            else if(arr[mid] == 1){
                mid++;
            }
            else{
                int temp = arr[mid];
                arr[mid] = arr[high];
                arr[high] = temp;
                high--;
            }
        }

        for(int num : arr){
            System.out.print(num + " ");
        }
    }
}
```
