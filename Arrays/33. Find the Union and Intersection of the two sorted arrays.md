# 33. Find Union and Intersection of Two Sorted Arrays

## 📌 Problem Statement

Given two **sorted arrays**, find their:

* **Union** → all unique elements from both arrays
* **Intersection** → common elements present in both arrays

---

## 📝 Input Format

* First line: Space-separated integers (sorted array A)
* Second line: Space-separated integers (sorted array B)

---

## 📤 Output Format

* Print the **union**
* Print the **intersection**

---

## 💡 Approach (Explanation)

### 🔑 Key Idea (Two Pointer Technique)

Since arrays are **already sorted**, use two pointers:

```text id="ptrs"
i → for array A  
j → for array B
```

---

## 🔁 Union Logic

1. Compare `A[i]` and `B[j]`
2. If equal → add once, move both pointers
3. If `A[i] < B[j]` → add `A[i]`, move `i`
4. Else → add `B[j]`, move `j`
5. Add remaining elements

---

## 🔁 Intersection Logic

1. If `A[i] == B[j]` → add element, move both
2. If `A[i] < B[j]` → move `i`
3. Else → move `j`

---

## 💻 Java Code

```java id="union-intersection"
import java.util.*;

class Main {
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);

        String[] aInput = sc.nextLine().split("\\s+");
        String[] bInput = sc.nextLine().split("\\s+");

        int[] A = new int[aInput.length];
        int[] B = new int[bInput.length];

        for(int i = 0; i < A.length; i++){
            A[i] = Integer.parseInt(aInput[i]);
        }
        for(int i = 0; i < B.length; i++){
            B[i] = Integer.parseInt(bInput[i]);
        }

        int i = 0, j = 0;

        List<Integer> union = new ArrayList<>();
        List<Integer> intersection = new ArrayList<>();

        // Union
        while(i < A.length && j < B.length){
            if(A[i] == B[j]){
                union.add(A[i]);
                i++; j++;
            }
            else if(A[i] < B[j]){
                union.add(A[i++]);
            }
            else{
                union.add(B[j++]);
            }
        }

        while(i < A.length) union.add(A[i++]);
        while(j < B.length) union.add(B[j++]);

        // Intersection
        i = 0; j = 0;
        while(i < A.length && j < B.length){
            if(A[i] == B[j]){
                intersection.add(A[i]);
                i++; j++;
            }
            else if(A[i] < B[j]){
                i++;
            }
            else{
                j++;
            }
        }

        // 🔥 Print Union (remove duplicates)
        Integer prev = null;
        System.out.print("Union: ");
        for(int x : union){
            if(prev == null || prev != x){
                System.out.print(x + " ");
            }
            prev = x;
        }

        System.out.println();

        // 🔥 Print Intersection (separate prev)
        Integer prev1 = null;
        System.out.print("Intersection: ");
        for(int x : intersection){
            if(prev1 == null || prev1 != x){
                System.out.print(x + " ");
            }
            prev1 = x;
        }
    }
}
```

---

## 🧪 Sample Input

```text id="input1"
1 2 3 4 5
2 3 5 7
```

---

## ✅ Sample Output

```text id="output1"
Union: 1 2 3 4 5 7
Intersection: 2 3 5
```

---

## 🧠 Explanation

* Union → all elements (no duplicates ideally)
* Intersection → only common elements

---

## 📊 Complexity Analysis

* Time Complexity: **O(n + m)**
* Space Complexity: **O(n + m)**

---

## 🔥 Key Points

* Works efficiently because arrays are sorted
* Two-pointer approach avoids extra sorting
* Handles both union and intersection in linear time
