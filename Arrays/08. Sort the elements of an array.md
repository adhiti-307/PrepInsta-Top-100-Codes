# 8. Sort the Elements of an Array

## 📌 Problem Statement

Write a program to sort the elements of a given array in **ascending order**.

---

## 📝 Input Format

* A single line containing space-separated integers (array elements)

---

## 📤 Output Format

* Print the sorted array

---

## 💡 Approach (Explanation)

1. Read the input array
2. Use a sorting method (built-in or manual)
3. Print the sorted array

---

## 💻 Java Code (Using Built-in Sort)

```java id="sort-array"
import java.util.*;

class Main {
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);

        String[] input = sc.nextLine().split("\\s+");
        int[] arr = new int[input.length];

        for(int i = 0; i < input.length; i++){
            arr[i] = Integer.parseInt(input[i]);
        }

        Arrays.sort(arr);

        for(int num : arr){
            System.out.print(num + " ");
        }
    }
}
```

---

## 🧪 Sample Input

```text id="input1"
5 2 8 7 1 6
```

---

## ✅ Sample Output

```text id="output1"
1 2 5 6 7 8
```

---

## 📊 Complexity Analysis

* Time Complexity: **O(n log n)**
* Space Complexity: **O(1)**

---

## 🔥 Key Points

* `Arrays.sort()` is efficient and commonly used
* Works for all integer values
* Sorting is a fundamental operation used in many problems
