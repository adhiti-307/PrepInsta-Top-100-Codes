# 4. Find Second Smallest Element in an Array

## 📌 Problem Statement

Write a program to find the **second smallest element** in a given array.

---

## 📝 Input Format

* A single line containing space-separated integers (array elements)

---

## 📤 Output Format

* Print the second smallest element
* If it does not exist (e.g., all elements are equal or only one element), print `Not Possible`

---

## 💡 Approach (Explanation)

1. Read the input array
2. Initialize two variables:

   * `min` = smallest element
   * `secondMin` = second smallest element
3. Traverse the array:

   * If current element < `min`:

     * Update `secondMin = min`
     * Update `min = current element`
   * Else if current element > `min` and < `secondMin`:

     * Update `secondMin`
4. After traversal:

   * If `secondMin` is unchanged → no second smallest exists
5. Print result

---

## 💻 Java Code

```java id="second-smallest"
import java.util.*;

class Main {
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);

        String[] input = sc.nextLine().split("\\s+");
        int[] arr = new int[input.length];

        for(int i = 0; i < input.length; i++){
            arr[i] = Integer.parseInt(input[i]);
        }

        int min = Integer.MAX_VALUE;
        int secondMin = Integer.MAX_VALUE;

        for(int num : arr){
            if(num < min){
                secondMin = min;
                min = num;
            }
            else if(num > min && num < secondMin){
                secondMin = num;
            }
        }

        if(secondMin == Integer.MAX_VALUE){
            System.out.println("Not Possible");
        } else {
            System.out.println(secondMin);
        }
    }
}
```

---

## 🧪 Sample Input

```text id="input1"
3 7 2 9 4
```

---

## ✅ Sample Output

```text id="output1"
3
```

---

## 📊 Complexity Analysis

* Time Complexity: **O(n)**
* Space Complexity: **O(1)**

---

## 🔥 Key Points

* Single traversal is enough
* Handles duplicates properly
* Use `Integer.MAX_VALUE` for initialization
