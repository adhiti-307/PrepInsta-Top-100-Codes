# 18. Count the Number of Even and Odd Elements in an Array

## 📌 Problem Statement

Write a program to count the number of **even** and **odd** elements in a given array.

---

## 📝 Input Format

* A single line containing space-separated integers (array elements)

---

## 📤 Output Format

* Print the count of even elements
* Print the count of odd elements

---

## 💡 Approach (Explanation)

1. Read the input array
2. Initialize two counters:

   * `evenCount = 0`
   * `oddCount = 0`
3. Traverse the array
4. For each element:

   * If `num % 2 == 0` → increment `evenCount`
   * Else → increment `oddCount`
5. Print both counts

---

## 💻 Java Code

```java id="even-odd-count"
import java.util.*;

class Main {
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);

        String[] input = sc.nextLine().split("\\s+");
        int[] arr = new int[input.length];

        for(int i = 0; i < input.length; i++){
            arr[i] = Integer.parseInt(input[i]);
        }

        int evenCount = 0;
        int oddCount = 0;

        for(int num : arr){
            if(num % 2 == 0){
                evenCount++;
            } else {
                oddCount++;
            }
        }

        System.out.println("Even: " + evenCount);
        System.out.println("Odd: " + oddCount);
    }
}
```

---

## 🧪 Sample Input

```text id="input1"
1 2 3 4 5 6
```

---

## ✅ Sample Output

```text id="output1"
Even: 3
Odd: 3
```

---

## 📊 Complexity Analysis

* Time Complexity: **O(n)**
* Space Complexity: **O(1)**

---

## 🔥 Key Points

* Use modulo operator `%` to check even/odd
* Single traversal is sufficient
* Works for both positive and negative integers
