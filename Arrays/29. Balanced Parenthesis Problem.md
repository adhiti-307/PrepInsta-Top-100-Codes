# 31. Balanced Parenthesis Problem

## 📌 Problem Statement

Write a program to check whether a given expression has **balanced parentheses**.

👉 An expression is balanced if:

* Every opening bracket has a corresponding closing bracket
* Brackets are properly nested

Supported brackets:

```text
( )   { }   [ ]
```

---

## 📝 Input Format

* A single line containing the expression (string)

---

## 📤 Output Format

* Print `Balanced` if parentheses are valid
* Otherwise print `Not Balanced`

---

## 💡 Approach (Explanation)

### 🔑 Key Idea: Use Stack

1. Traverse each character in the string
2. If it is an **opening bracket** → push to stack
3. If it is a **closing bracket**:

   * If stack is empty → not balanced
   * Pop top element and check if it matches
4. After traversal:

   * If stack is empty → balanced
   * Else → not balanced

---

## 💻 Java Code

```java
import java.util.*;

class Main {
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);
        String str = sc.nextLine();

        Stack<Character> stack = new Stack<>();
        boolean isBalanced = true;

        for(char ch : str.toCharArray()){
            
            if(ch == '(' || ch == '{' || ch == '['){
                stack.push(ch);
            }
            else if(ch == ')' || ch == '}' || ch == ']'){
                
                if(stack.isEmpty()){
                    isBalanced = false;
                    break;
                }

                char top = stack.pop();

                if((ch == ')' && top != '(') ||
                   (ch == '}' && top != '{') ||
                   (ch == ']' && top != '[')){
                    isBalanced = false;
                    break;
                }
            }
        }

        if(!stack.isEmpty()){
            isBalanced = false;
        }

        System.out.println(isBalanced ? "Balanced" : "Not Balanced");
    }
}
```

## 🧠 Dry Run (Important)

Input:

```text
{[()]}
```

Steps:

```text
{ → push
[ → push
( → push
) → matches ( → pop
] → matches [ → pop
} → matches { → pop
```

👉 Stack empty → Balanced ✅

---

## 📊 Complexity Analysis

* Time Complexity: **O(n)**
* Space Complexity: **O(n)**

---

## 🔥 Key Points

* Stack ensures correct order of matching
* Check empty stack before popping
* Must match corresponding bracket types
