# 6. Reverse an Array

## 📌 Problem Statement

Write a program to reverse the elements of a given array.

---

## 📝 Input Format

* A single line containing space-separated integers (array elements)

---

## 📤 Output Format

* Print the reversed array

---

## 💡 Approach (Explanation)

1. Read the input array
2. Use two pointers:

   * `left = 0` (start of array)
   * `right = n - 1` (end of array)
3. Swap elements at `left` and `right`
4. Move:

   * `left++`
   * `right--`
5. Repeat until `left < right`
6. Print the reversed array

---

## 💻 Java Code

```java id="reverse-array"
import java.util.*;

class Main {
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);

        String[] input = sc.nextLine().split("\\s+");
        int[] arr = new int[input.length];

        for(int i = 0; i < input.length; i++){
            arr[i] = Integer.parseInt(input[i]);
        }

        int left = 0;
        int right = arr.length - 1;

        while(left < right){
            int temp = arr[left];
            arr[left] = arr[right];
            arr[right] = temp;

            left++;
            right--;
        }

        for(int num : arr){
            System.out.print(num + " ");
        }
    }
}
```

## 💻 Java Code

```java id="reverse-array"
import java.util.*;

class Main {
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);

        String[] input = sc.nextLine().split("\\s+");
        int[] arr = new int[input.length];

        for(int i = 0; i < input.length; i++){
            arr[i] = Integer.parseInt(input[i]);
        }

        for(int i = arr.length - 1; i >= 0; i--){
            System.out.print(arr[i] + " ");
        }
    }
}
```

---
## 🧪 Sample Input

```text id="input1"
1 2 3 4 5
```

* Uses two-pointer technique
* In-place reversal (no extra space required)
* Efficient and commonly asked in interviews
