# 34. Move All Negative Elements to One Side of the Array

## 📌 Problem Statement

Write a program to move all the **negative elements** to one side of the array.
👉 Order of elements does **not matter**.

---

## 📝 Input Format

* A single line containing space-separated integers (array elements)

---

## 📤 Output Format

* Print the modified array

---

## 💡 Approach (Explanation)

### 🔑 Key Idea (Two Pointer Technique)

Use two pointers:

```text id="s3t8xq"
left  → start of array  
right → end of array  
```

---

### 🔁 Steps

1. If `arr[left]` is negative → move `left++`
2. If `arr[right]` is positive → move `right--`
3. Else → swap `arr[left]` and `arr[right]`

Repeat until `left < right`

---
## 💻 Java Code

```java id="move-negatives"
import java.util.*; 

class Main { 
    public static void main(String[] args){ 
        Scanner sc = new Scanner(System.in); 
        String[] input = sc.nextLine().split("\\s+"); 
        
        int[] arr = new int[input.length]; 
        for(int i = 0; i < input.length; i++){ 
            arr[i] = Integer.parseInt(input[i]); 
        } 
        
        int left = 0; 
        int right = arr.length - 1; 
        
        while(left < right){ 
            if(arr[left] < 0){
                left++; 
            } else if(arr[right] >= 0){
                right--; 
            } else{ 
                int temp = arr[left]; 
                arr[left] = arr[right]; 
                arr[right] = temp; left++; right--; 
            } 
        } 
        
        for(int num : arr){ 
            System.out.print(num + " "); 
        } 
    } 
}
```

## 💻 Java Code

```java id="move-negatives"
import java.util.*;

class Main{
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);
        String[] parts = sc.nextLine().split("\\s+");

        int[] arr = new int[parts.length];
        for(int i = 0; i < parts.length; i++){
            arr[i] = Integer.parseInt(parts[i]);
        }

        int low = 0, mid = 0;
        int high = arr.length - 1;

        while(mid <= high){
            if(arr[mid] < 0){
                int temp = arr[low];
                arr[low] = arr[mid];
                arr[mid] = temp;
                low++;
                mid++;
            }
            else{
                int temp = arr[mid];
                arr[mid] = arr[high];
                arr[high] = temp;
                high--;   // 
            }
        }

        for(int x : arr){
            System.out.print(x + " ");
        }
    }
}
```

## 🧠 Explanation

* Negative elements moved to left
* Positive elements moved to right
* Order may change (not stable)

---

## 📊 Complexity Analysis

* Time Complexity: **O(n)**
* Space Complexity: **O(1)**

---

## 🔥 Key Points

* In-place solution (no extra space)
* Uses two-pointer technique
* Order is not preserved

---

## ⚡ Alternative (Stable Approach)
