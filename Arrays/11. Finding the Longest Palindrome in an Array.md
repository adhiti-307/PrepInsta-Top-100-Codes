# 11. Find the Longest Palindrome in an Array

## 📌 Problem Statement

Write a program to find the **longest palindrome string** present in a given array of strings.

A palindrome is a string that reads the same forward and backward.

---

## 📝 Input Format

* A single line containing space-separated strings

---

## 📤 Output Format

* Print the longest palindrome string
* If no palindrome exists, print `Not Found`

---

## 💡 Approach (Explanation)

1. Read the input array of strings
2. Initialize:

   * `maxLen = 0`
   * `result = ""`
3. Traverse each string:

   * Check if it is a palindrome
   * If yes and its length > `maxLen`, update result
4. After traversal:

   * If result is empty → print `Not Found`
   * Else print result

---

## 💻 Java Code

```java id="longest-palindrome-array"
import java.util.*;

class Main {

    static boolean isPalindrome(String s){
        int left = 0, right = s.length() - 1;

        while(left < right){
            if(s.charAt(left) != s.charAt(right)){
                return false;
            }
            left++;
            right--;
        }
        return true;
    }

    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);

        String[] arr = sc.nextLine().split("\\s+");

        int maxLen = 0;
        String result = "";

        for(String str : arr){
            if(isPalindrome(str)){
                if(str.length() > maxLen){
                    maxLen = str.length();
                    result = str;
                }
            }
        }

        if(result.equals("")){
            System.out.println("Not Found");
        } else {
            System.out.println(result);
        }
    }
}
```

---

## 🧪 Sample Input

```text id="input1"
madam racecar apple level noon
```

---

## ✅ Sample Output

```text id="output1"
racecar
```

---

## 📊 Complexity Analysis

* Time Complexity: **O(n × k)**

  * `n` = number of strings
  * `k` = average length of each string
* Space Complexity: **O(1)**

---

## 🔥 Key Points

* Use two-pointer technique for palindrome checking
* Track longest length during traversal
* Works for both even and odd length palindromes
