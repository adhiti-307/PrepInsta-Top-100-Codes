# 17. Find Maximum Scalar Product of Two Vectors

## 📌 Problem Statement

Write a program to find the **maximum scalar product** of two vectors.

The scalar product of two vectors is:

```text id="eqn1"
sum = a1*b1 + a2*b2 + ... + an*bn
```

---

## 📝 Input Format

* First line: Space-separated integers (vector A)
* Second line: Space-separated integers (vector B)

---

## 📤 Output Format

* Print the maximum scalar product

---

## 💡 Approach (Explanation)

To maximize the scalar product:

1. Sort both arrays in **ascending order**
2. Multiply corresponding elements
3. Add all products

👉 This pairs:

```text id="intuition"
small × small
large × large
```

which gives the **maximum possible sum**

---

## 💻 Java Code

```java id="max-scalar-product"
import java.util.*;

class Main {
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);

        String[] inputA = sc.nextLine().split("\\s+");
        String[] inputB = sc.nextLine().split("\\s+");

        int n = inputA.length;

        int[] A = new int[n];
        int[] B = new int[n];

        for(int i = 0; i < n; i++){
            A[i] = Integer.parseInt(inputA[i]);
            B[i] = Integer.parseInt(inputB[i]);
        }

        Arrays.sort(A);
        Arrays.sort(B);

        int sum = 0;

        for(int i = 0; i < n; i++){
            sum += A[i] * B[i];
        }

        System.out.println(sum);
    }
}
```

---

## 🧪 Sample Input

```text id="input1"
1 3 -5
-2 4 1
```

---

## ✅ Sample Output

```text id="output1"
23
```

---

## 🧠 Explanation

After sorting:

```text id="exp"
A → [-5, 1, 3]
B → [-2, 1, 4]
```

Multiply:

```text id="calc"
(-5 * -2) + (1 * 1) + (3 * 4)
= 10 + 1 + 12
= 23
```

---

## 📊 Complexity Analysis

* Time Complexity: **O(n log n)**
* Space Complexity: **O(1)**

---

## 🔥 Key Points

* Sort both arrays in the **same order**
* Pair similar magnitude values
* Greedy strategy ensures maximum result
