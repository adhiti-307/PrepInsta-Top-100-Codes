# 2. Find Smallest Element in an Array

## 📌 Problem Statement

Write a program to find the **smallest element** in a given array.

---

## 📝 Input Format

* A single line containing space-separated integers (array elements)

---

## 📤 Output Format

* Print the smallest element in the array

---

## 💡 Approach (Explanation)

1. Read the input array
2. Initialize a variable `min` with the first element
3. Traverse the array
4. If any element is smaller than `min`, update `min`
5. Print `min`

---

## 💻 Java Code

```java id="smallest-array"
import java.util.*;

class Main {
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);

        String[] input = sc.nextLine().split("\\s+");
        int[] arr = new int[input.length];

        for(int i = 0; i < input.length; i++){
            arr[i] = Integer.parseInt(input[i]);
        }

        int min = arr[0];

        for(int i = 1; i < arr.length; i++){
            if(arr[i] < min){
                min = arr[i];
            }
        }

        System.out.println(min);
    }
}
```

---

## 🧪 Sample Input

```text id="input1"
3 7 2 9 4
```

---

## ✅ Sample Output

```text id="output1"
2
```

---

## 📊 Complexity Analysis

* Time Complexity: **O(n)**
* Space Complexity: **O(1)**
