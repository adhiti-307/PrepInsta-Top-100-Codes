# 13. Find Repeating Elements in an Array

## 📌 Problem Statement

Write a program to find all **repeating elements** in a given array.

---

## 📝 Input Format

* A single line containing space-separated integers (array elements)

---

## 📤 Output Format

* Print all elements that appear more than once
* If no repeating element exists, print `No Repeating Elements`

---

## 💡 Approach (Explanation)

1. Read the input array
2. Use a `HashMap` to store frequency of each element
3. Traverse the array:

   * Update frequency using `getOrDefault()`
4. Traverse the map:

   * If frequency > 1 → print the element
5. If no such element found → print message

---

## 💻 Java Code

```java id="repeating-elements"
import java.util.*;

class Main {
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);

        String[] input = sc.nextLine().split("\\s+");
        int[] arr = new int[input.length];

        for(int i = 0; i < input.length; i++){
            arr[i] = Integer.parseInt(input[i]);
        }

        HashMap<Integer, Integer> map = new HashMap<>();

        // Count frequency
        for(int num : arr){
            map.put(num, map.getOrDefault(num, 0) + 1);
        }

        boolean found = false;

        // Print repeating elements
        for(Map.Entry<Integer, Integer> entry : map.entrySet()){
            if(entry.getValue() > 1){
                System.out.print(entry.getKey() + " ");
                found = true;
            }
        }

        if(!found){
            System.out.println("No Repeating Elements");
        }
    }
}
```

---

## 🧪 Sample Input

```text id="input1"
1 2 2 3 1 4 2
```

---

## ✅ Sample Output

```text id="output1"
1 2
```

---

## 📊 Complexity Analysis

* Time Complexity: **O(n)**
* Space Complexity: **O(n)**

---

## 🔥 Key Points

* Uses `HashMap` for efficient counting
* Prints elements with frequency > 1
* Works for all integer values
