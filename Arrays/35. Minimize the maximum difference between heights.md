# 35. Minimize the Maximum Difference Between Heights

## 📌 Problem Statement

Given an array of heights and an integer `k`, you can **increase or decrease each element by `k` (only once)**.

👉 Find the **minimum possible difference between the maximum and minimum heights** after modification.

---

## 📝 Input Format

* First line: Space-separated integers (heights)
* Second line: Integer `k`

---

## 📤 Output Format

* Print the minimized difference

---

## 💡 Approach (Explanation)

### 🔑 Key Idea

1. Sort the array
2. Initial difference:

   ```text
   maxDiff = arr[n-1] - arr[0]
   ```
3. Try to minimize this by:

   * Increasing smaller elements by `k`
   * Decreasing larger elements by `k`

---

### 🧠 Core Logic

After sorting:

```text
Small elements → +k  
Large elements → -k
```

We check every possible split:

```text
newMin = min(arr[0] + k, arr[i] - k)
newMax = max(arr[i-1] + k, arr[n-1] - k)
```

---

## 💻 Java Code

```java id="min-height-diff"
import java.util.*;

class Main {
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);

        String[] input = sc.nextLine().split("\\s+");
        int k = sc.nextInt();

        int n = input.length;
        int[] arr = new int[n];

        for(int i = 0; i < n; i++){
            arr[i] = Integer.parseInt(input[i]);
        }

        Arrays.sort(arr);

        int ans = arr[n-1] - arr[0];

        int smallest = arr[0] + k;
        int largest = arr[n-1] - k;

        for(int i = 1; i < n; i++){

            if(arr[i] - k < 0) continue;

            int minVal = Math.min(smallest, arr[i] - k);
            int maxVal = Math.max(largest, arr[i-1] + k);

            ans = Math.min(ans, maxVal - minVal);
        }

        System.out.println(ans);
    }
}
```

---

## 🧪 Sample Input

```text id="input1"
1 5 8 10
2
```

---

## ✅ Sample Output

```text id="output1"
5
```

---

## 🧠 Explanation

After modification:

```text id="exp1"
[1+2, 5+2, 8-2, 10-2]
→ [3, 7, 6, 8]
```

Difference:

```text id="exp2"
8 - 3 = 5
```

---

## 📊 Complexity Analysis

* Time Complexity: **O(n log n)** (sorting)
* Space Complexity: **O(1)**

---

## 🔥 Key Points

* Sorting is essential
* Greedy approach
* Try all partitions
* Skip negative heights

---

## ⚡ One-line memory trick

```text id="mem1"
Raise small, lower big → minimize gap
```
