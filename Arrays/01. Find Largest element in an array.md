# 01. Find Largest Element in an Array

## 📌 Problem Statement

Write a program to find the **largest element** in a given array.

---

## 📝 Input Format

* First line: Integer `n` (size of array)
* Second line: `n` space-separated integers

---

## 📤 Output Format

* Print the largest element in the array

---

## 🔍 Constraints

* `1 ≤ n ≤ 10^5`
* Array elements can be positive or negative integers

---

## 💡 Approach (Explanation)

1. Read the size of the array and its elements
2. Initialize a variable `max` with the first element
3. Traverse the array from index `1` to `n-1`
4. For each element:

   * If current element > `max`, update `max`
5. After traversal, `max` will store the largest element

---

## 💻 Java Code

```java id="largest-element"
import java.util.*;

class Main {
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);

        int n = sc.nextInt();
        int[] arr = new int[n];

        for(int i = 0; i < n; i++){
            arr[i] = sc.nextInt();
        }

        int max = arr[0];

        for(int i = 1; i < n; i++){
            if(arr[i] > max){
                max = arr[i];
            }
        }

        System.out.println(max);
    }
}
```

---

## 🧪 Sample Input

```text id="input1"
5
3 7 2 9 4
```

---

## ✅ Sample Output

```text id="output1"
9
```

---

## 📊 Complexity Analysis

* Time Complexity: **O(n)**
* Space Complexity: **O(1)**

---

## 🔥 Key Points

* Only one traversal is needed
* Works for both positive and negative numbers
* Initialize `max` with first element (important for negative arrays)
