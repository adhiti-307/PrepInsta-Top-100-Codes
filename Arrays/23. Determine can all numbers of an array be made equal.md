# 23. Determine if All Numbers of an Array Can Be Made Equal

## 📌 Problem Statement

Write a program to check whether all elements of an array can be made **equal**.

👉 Allowed operations:

* Multiply any element by **2**
* Multiply any element by **3**

---

## 📝 Input Format

* A single line containing space-separated integers (array elements)

---

## 📤 Output Format

* Print `Yes` if all elements can be made equal
* Otherwise print `No`

---

## 💡 Approach (Explanation)

### 🔑 Key Idea

Any number can be expressed as:

```text id="eq1"
n = 2^a × 3^b × (other prime factors)
```

* We can change powers of **2 and 3**
* But **other prime factors cannot be changed**

👉 So:

1. Remove all factors of **2** from every number
2. Remove all factors of **3** from every number
3. Compare all resulting values
4. If all are equal → `Yes`, else → `No`

---

## 💻 Java Code

```java id="given-code"
import java.util.*;

public class Main
{
    public static boolean make_equal(int arr[], int n)
    {
        for (int i = 0; i < n; i++)
        {
            // Divide number by 2
            while (arr[i] % 2 == 0)
                arr[i] /= 2;

            // Divide number by 3
            while (arr[i] % 3 == 0)
                arr[i] /= 3;
        }

        // Compare remaining values
        for (int i = 1; i < n; i++)
        {
            if (arr[i] != arr[0])
                return false;
        }

        return true;
    }

    public static void main(String[] args)
    {
        Scanner sc = new Scanner(System.in);

        String[] input = sc.nextLine().split("\\s+");
        int n = input.length;
        int[] arr = new int[n];

        for (int i = 0; i < n; i++)
        {
            arr[i] = Integer.parseInt(input[i]);
        }

        if (make_equal(arr, n))
            System.out.println("Yes");
        else
            System.out.println("No");
    }
}
```

* Remove factors of **2 and 3** from each number
* Remaining values must be identical
* Uses in-place modification of array
