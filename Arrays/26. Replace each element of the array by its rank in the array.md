# 26. Replace Each Element of the Array by Its Rank

## 📌 Problem Statement

Write a program to replace each element of an array with its **rank**.

👉 Rank is defined as the position of the element in the **sorted unique array**
👉 The smallest element gets rank **1**

---

## 📝 Input Format

* A single line containing space-separated integers (array elements)

---

## 📤 Output Format

* Print the array after replacing each element with its rank

---

## 💡 Approach (Explanation)

1. Read the input array
2. Create a copy of the array
3. Sort the copied array
4. Assign ranks:

   * Use a `HashMap`
   * Traverse sorted array
   * Assign rank only for unique elements
5. Replace each element in original array using the map

---

## 💻 Java Code

```java id="rank-array"
import java.util.*;

class Main {
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);

        String[] input = sc.nextLine().split("\\s+");
        int n = input.length;

        int[] arr = new int[n];
        for(int i = 0; i < n; i++){
            arr[i] = Integer.parseInt(input[i]);
        }

        // Step 1: Copy array
        int[] temp = arr.clone();

        // Step 2: Sort copy
        Arrays.sort(temp);

        // Step 3: Assign ranks
        HashMap<Integer, Integer> map = new HashMap<>();
        int rank = 1;

        for(int num : temp){
            if(!map.containsKey(num)){
                map.put(num, rank++);
            }
        }

        // Step 4: Replace elements with rank
        for(int i = 0; i < n; i++){
            arr[i] = map.get(arr[i]);
        }

        // Print result
        for(int num : arr){
            System.out.print(num + " ");
        }
    }
}
```

---

## 🧪 Sample Input

```text id="input1"
20 15 26 2 98 6
```

---

## ✅ Sample Output

```text id="output1"
4 3 5 1 6 2
```

---

## 🧠 Explanation

Sorted unique elements:

```text id="exp1"
[2, 6, 15, 20, 26, 98]
```

Assign ranks:

```text id="exp2"
2→1, 6→2, 15→3, 20→4, 26→5, 98→6
```

Replace:

```text id="exp3"
20→4, 15→3, 26→5, 2→1, 98→6, 6→2
```

---

## 📊 Complexity Analysis

* Time Complexity: **O(n log n)**
* Space Complexity: **O(n)**

---

## 🔥 Key Points

* Ranking is based on sorted order
* Duplicate elements get the same rank
* Use `HashMap` for fast lookup
