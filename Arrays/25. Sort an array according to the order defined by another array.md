# 25. Sort an Array According to the Order Defined by Another Array

## 📌 Problem Statement

Given two arrays **A[]** and **B[]**, sort array **A** such that:

* The **relative order of elements** in A follows the order given in B
* Elements present in B should appear first in A
* Remaining elements (not present in B) should be placed at the end

---

## 📝 Input Format

* First line: Space-separated integers (array A)
* Second line: Space-separated integers (array B)

---

## 📤 Output Format

* Print the modified array A after sorting

---

## 💡 Approach (Explanation)

1. Store frequency of elements of array **A** using a `HashMap`
2. Traverse array **B**:

   * For each element in B:

     * If present in map → print it as many times as its frequency
     * Remove it from the map
3. For remaining elements in map:

   * Print them at the end (in any order or sorted if required)

---

## 💻 Java Code

```java id="relative-sort-clean"
import java.util.*;

class Main {
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);

        String[] inputA = sc.nextLine().split("\\s+");
        String[] inputB = sc.nextLine().split("\\s+");

        int[] A = new int[inputA.length];
        int[] B = new int[inputB.length];

        for(int i = 0; i < inputA.length; i++){
            A[i] = Integer.parseInt(inputA[i]);
        }

        for(int i = 0; i < inputB.length; i++){
            B[i] = Integer.parseInt(inputB[i]);
        }

        HashMap<Integer, Integer> map = new HashMap<>();

        // Step 1: Count frequency of A
        for(int num : A){
            map.put(num, map.getOrDefault(num, 0) + 1);
        }

        // Step 2: Print elements based on B
        for(int num : B){
            if(map.containsKey(num)){
                int count = map.get(num);

                while(count-- > 0){
                    System.out.print(num + " ");
                }

                map.remove(num);
            }
        }

        // Step 3: Print remaining elements
        for(Map.Entry<Integer, Integer> entry : map.entrySet()){
            int num = entry.getKey();
            int count = entry.getValue();

            while(count-- > 0){
                System.out.print(num + " ");
            }
        }
    }
}
```

---

## 🧪 Sample Input

```text id="input1"
2 1 2 5 7 1 9 3 6 8 8
2 1 8 3
```

---

## ✅ Sample Output

```text id="output1"
2 2 1 1 8 8 3 5 7 9 6
```

---

## 🧠 Explanation

* Elements from B → `2 1 8 3` come first
* Remaining elements → `5 7 9 6` printed at the end

---

## 📊 Complexity Analysis

* Time Complexity: **O(n + m)**
* Space Complexity: **O(n)**

---

## 🔥 Key Points

* Maintain order defined by second array
* Use `HashMap` for frequency tracking
* Remaining elements can be printed as is or sorted if required
