# 16. Find Minimum Scalar Product of Two Vectors

## 📌 Problem Statement

Write a program to find the **minimum scalar product** of two vectors.

The scalar product of two vectors is defined as:

```text
sum = a1*b1 + a2*b2 + ... + an*bn
```

---

## 📝 Input Format

* First line: Space-separated integers (vector A)
* Second line: Space-separated integers (vector B)

---

## 📤 Output Format

* Print the minimum scalar product

---

## 💡 Approach (Explanation)

To minimize the scalar product:

1. Sort vector **A in ascending order**
2. Sort vector **B in descending order**
3. Multiply corresponding elements and sum them

👉 This pairing ensures the smallest values multiply with largest values, minimizing the result.

---

## 💻 Java Code

```java id="min-scalar-product"
import java.util.*;

class Main {
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);

        String[] inputA = sc.nextLine().split("\\s+");
        String[] inputB = sc.nextLine().split("\\s+");

        int n = inputA.length;

        int[] A = new int[n];
        int[] B = new int[n];

        for(int i = 0; i < n; i++){
            A[i] = Integer.parseInt(inputA[i]);
            B[i] = Integer.parseInt(inputB[i]);
        }

        Arrays.sort(A); // ascending
        Arrays.sort(B); // ascending

        // reverse B for descending
        for(int i = 0; i < n/2; i++){
            int temp = B[i];
            B[i] = B[n-1-i];
            B[n-1-i] = temp;
        }

        int sum = 0;

        for(int i = 0; i < n; i++){
            sum += A[i] * B[i];
        }

        System.out.println(sum);
    }
}
```

---

## 🧪 Sample Input

```text id="input1"
1 3 -5
-2 4 1
```

---

## ✅ Sample Output

```text id="output1"
-25
```

---

## 🧠 Explanation

After sorting:

```text
A → [-5, 1, 3]
B → [4, 1, -2]
```

Multiply:

```text
(-5 * 4) + (1 * 1) + (3 * -2)
= -20 + 1 - 6
= -25
```

---

## 📊 Complexity Analysis

* Time Complexity: **O(n log n)** (sorting)
* Space Complexity: **O(1)**

---

## 🔥 Key Points

* Sort one array ascending and the other descending
* Greedy strategy ensures minimum result
* Common greedy problem in placements
