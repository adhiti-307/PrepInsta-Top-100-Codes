# 28. Rotation of Elements of an Array (Left and Right)

## 📌 Problem Statement

Write a program to rotate the elements of an array:

* **Left Rotation** → shift elements to the left
* **Right Rotation** → shift elements to the right

---

## 📝 Input Format

* First line: Space-separated integers (array elements)
* Second line: Integer `k` (number of rotations)

---

## 📤 Output Format

* Print the array after left rotation
* Print the array after right rotation

---

## 💡 Approach (Explanation)

### 🔑 Key Idea

Use **Reversal Algorithm** (efficient, O(n))

---

## 🔹 Left Rotation by k

Steps:

1. Reverse first `k` elements
2. Reverse remaining `n-k` elements
3. Reverse entire array

---

## 🔹 Right Rotation by k

Steps:

1. Reverse last `k` elements
2. Reverse first `n-k` elements
3. Reverse entire array

---

## 💻 Java Code

```java id="array-rotation"
import java.util.*;

class Main {

    static void reverse(int[] arr, int start, int end){
        while(start < end){
            int temp = arr[start];
            arr[start] = arr[end];
            arr[end] = temp;
            start++;
            end--;
        }
    }

    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);

        String[] input = sc.nextLine().split("\\s+");
        int k = sc.nextInt();

        int n = input.length;
        int[] arr = new int[n];

        for(int i = 0; i < n; i++){
            arr[i] = Integer.parseInt(input[i]);
        }

        k = k % n;

        // LEFT ROTATION
        int[] leftArr = arr.clone();

        reverse(leftArr, 0, k - 1);
        reverse(leftArr, k, n - 1);
        reverse(leftArr, 0, n - 1);

        // RIGHT ROTATION
        int[] rightArr = arr.clone();

        reverse(rightArr, n - k, n - 1);
        reverse(rightArr, 0, n - k - 1);
        reverse(rightArr, 0, n - 1);

        // Print results
        System.out.print("Left Rotation: ");
        for(int num : leftArr){
            System.out.print(num + " ");
        }

        System.out.println();

        System.out.print("Right Rotation: ");
        for(int num : rightArr){
            System.out.print(num + " ");
        }
    }
}
```

## 📊 Complexity Analysis

* Time Complexity: **O(n)**
* Space Complexity: **O(1)**

---

## 🔥 Key Points

* Use `k = k % n` to handle large rotations
* Reversal method is optimal
* Avoid shifting one-by-one (inefficient)
