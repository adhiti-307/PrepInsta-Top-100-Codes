# 10. Sort Elements of an Array by Frequency

## 📌 Problem Statement

Write a program to sort the elements of an array based on their **frequency**.

* Elements with **higher frequency** should come first
* If two elements have the same frequency, sort them in **ascending order**

---

## 📝 Input Format

* A single line containing space-separated integers (array elements)

---

## 📤 Output Format

* Print the array sorted by frequency

---

## 💡 Approach (Explanation)

1. Read the input array
2. Use a `HashMap` to store frequency of each element
3. Convert array into a list (for custom sorting)
4. Sort the list using a custom comparator:

   * Compare frequency (descending)
   * If equal → compare value (ascending)
5. Print the sorted list

---

## 💻 Java Code

```java id="sort-by-frequency"
import java.util.*;

class Main {
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);

        String[] input = sc.nextLine().split("\\s+");
        int[] arr = new int[input.length];

        for(int i = 0; i < input.length; i++){
            arr[i] = Integer.parseInt(input[i]);
        }

        HashMap<Integer, Integer> freqMap = new HashMap<>();

        // Count frequency
        for(int num : arr){
            freqMap.put(num, freqMap.getOrDefault(num, 0) + 1);
        }

        // Convert to list
        List<Integer> list = new ArrayList<>();
        for(int num : arr){
            list.add(num);
        }

        // Sort based on frequency and value
        Collections.sort(list, (a, b) -> {
            int freqA = freqMap.get(a);
            int freqB = freqMap.get(b);

            if(freqA != freqB){
                return freqB - freqA; // higher frequency first
            }
            return a - b; // smaller value first
        });

        // Print result
        for(int num : list){
            System.out.print(num + " ");
        }
    }
}
```

---

## 🧪 Sample Input

```text id="input1"
1 2 2 3 1 4 2
```

---

## ✅ Sample Output

```text id="output1"
2 2 2 1 1 3 4
```

---

## 📊 Complexity Analysis

* Time Complexity: **O(n log n)** (due to sorting)
* Space Complexity: **O(n)**

---

## 🔥 Key Points

* Use `HashMap` to count frequencies
* Custom comparator controls sorting logic
* Handles both frequency and value ordering
