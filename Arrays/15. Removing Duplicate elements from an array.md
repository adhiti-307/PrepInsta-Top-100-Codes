# 15. Remove Duplicate Elements from an Array

## 📌 Problem Statement

Write a program to remove **duplicate elements** from a given array and print only **unique elements**.

---

## 📝 Input Format

* A single line containing space-separated integers (array elements)

---

## 📤 Output Format

* Print the array after removing duplicates

---

## 💡 Approach (Explanation)

1. Read the input array
2. Use a `LinkedHashSet` to store elements:

   * It removes duplicates
   * It preserves insertion order
3. Traverse the array and add elements to the set
4. Print elements from the set

---

## 💻 Java Code

```java id="remove-duplicates"
import java.util.*;

class Main {
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);

        String[] input = sc.nextLine().split("\\s+");
        int[] arr = new int[input.length];

        for(int i = 0; i < input.length; i++){
            arr[i] = Integer.parseInt(input[i]);
        }

        LinkedHashSet<Integer> set = new LinkedHashSet<>();

        for(int num : arr){
            set.add(num);
        }

        for(int num : set){
            System.out.print(num + " ");
        }
    }
}
```

---

## 🧪 Sample Input

```text id="input1"
1 2 2 3 1 4 5
```

---

## ✅ Sample Output

```text id="output1"
1 2 3 4 5
```

---

## 📊 Complexity Analysis

* Time Complexity: **O(n)**
* Space Complexity: **O(n)**

---

## 🔥 Key Points

* `LinkedHashSet` removes duplicates and maintains order
* Efficient and simple approach
* No need for sorting
