# 27. Find Equilibrium Index of an Array

## 📌 Problem Statement

Write a program to find the **equilibrium index** of an array.

👉 An index `i` is called an equilibrium index if:

```text
Sum of elements on left of i = Sum of elements on right of i
```

---

## 📝 Input Format

* A single line containing space-separated integers (array elements)

---

## 📤 Output Format

* Print the equilibrium index
* If no such index exists, print `-1`

---

## 💡 Approach (Explanation)

### 🔑 Key Idea

Instead of recalculating left and right sums every time:

1. Compute **total sum** of the array
2. Initialize `leftSum = 0`
3. Traverse the array:

   * Subtract current element from total → gives **right sum**
   * Compare:

     ```text
     leftSum == rightSum
     ```
   * If equal → equilibrium index found
   * Add current element to `leftSum`

---

## 💻 Java Code

```java id="equilibrium-index"
import java.util.*;

class Main {
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);

        String[] input = sc.nextLine().split("\\s+");
        int n = input.length;
        int[] arr = new int[n];

        for(int i = 0; i < n; i++){
            arr[i] = Integer.parseInt(input[i]);
        }

        int totalSum = 0;
        for(int num : arr){
            totalSum += num;
        }

        int leftSum = 0;

        for(int i = 0; i < n; i++){
            totalSum -= arr[i]; // right sum

            if(leftSum == totalSum){
                System.out.println(i);
                return;
            }

            leftSum += arr[i];
        }

        System.out.println(-1);
    }
}
```

---

## 🧪 Sample Input

```text id="input1"
-7 1 5 2 -4 3 0
```

---

## ✅ Sample Output

```text id="output1"
3
```

---

## 🧠 Explanation

At index `3`:

```text
Left sum  = -7 + 1 + 5 = -1  
Right sum = -4 + 3 + 0 = -1  
```

👉 Equal → equilibrium index = **3**

---

## 📊 Complexity Analysis

* Time Complexity: **O(n)**
* Space Complexity: **O(1)**

---

## 🔥 Key Points

* Use running sum instead of nested loops
* Avoid recalculating sums repeatedly
* Efficient single-pass solution
