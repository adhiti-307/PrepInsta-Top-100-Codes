# 21. Check if Two Arrays are Disjoint or Not

## 📌 Problem Statement

Write a program to check whether two arrays are **disjoint** or not.

Two arrays are said to be **disjoint** if they have **no common elements**.

---

## 📝 Input Format

* First line: Space-separated integers (array A)
* Second line: Space-separated integers (array B)

---

## 📤 Output Format

* Print `Disjoint` if no common element exists
* Otherwise print `Not Disjoint`

---

## 💡 Approach (Explanation)

1. Read both arrays
2. Store elements of the first array in a `HashSet`
3. Traverse the second array:

   * If any element exists in the set → arrays are not disjoint
4. If no common element is found → arrays are disjoint

---

## 💻 Java Code

```java id="disjoint-arrays"
import java.util.*;

class Main {
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);

        String[] inputA = sc.nextLine().split("\\s+");
        String[] inputB = sc.nextLine().split("\\s+");

        HashSet<Integer> set = new HashSet<>();

        for(String s : inputA){
            set.add(Integer.parseInt(s));
        }

        boolean isDisjoint = true;

        for(String s : inputB){
            int num = Integer.parseInt(s);
            if(set.contains(num)){
                isDisjoint = false;
                break;
            }
        }

        if(isDisjoint){
            System.out.println("Disjoint");
        } else {
            System.out.println("Not Disjoint");
        }
    }
}
```

---

## 🧪 Sample Input

```text id="input1"

```

## ✅ Sample Output

