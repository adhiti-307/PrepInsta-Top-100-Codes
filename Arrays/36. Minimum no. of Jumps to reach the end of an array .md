# 36. Minimum Number of Jumps to Reach End of Array

## 📌 Problem Statement

Given an array where each element represents the **maximum jump length** from that position, find the **minimum number of jumps** required to reach the last index.

👉 If it is not possible to reach the end, print `-1`

---

## 📝 Input Format

* A single line containing space-separated integers (array elements)

---

## 📤 Output Format

* Print the minimum number of jumps

---

## 💡 Approach (Explanation)

### 🔑 Key Idea (Greedy)

Instead of trying all paths (DP/recursion), we track:

```text id="a1"
maxReach → farthest index we can reach  
steps    → steps left in current jump  
jumps    → number of jumps taken  
```

---

## 🔁 Logic

1. Initialize:

   ```text id="a2"
   maxReach = arr[0]  
   steps = arr[0]  
   jumps = 1  
   ```

2. Traverse array from index 1:

   * Update:

     ```text
     maxReach = max(maxReach, i + arr[i])
     steps--
     ```

   * If `steps == 0`:

     * Make a jump:

       ```text
       jumps++
       ```
     * If `i >= maxReach` → return `-1`
     * Reset:

       ```text
       steps = maxReach - i
       ```

3. If last index reached → return jumps

---

## 💻 Java Code

```java id="min-jumps"
import java.util.*;

class Main {
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);

        String[] input = sc.nextLine().split("\\s+");
        int n = input.length;

        int[] arr = new int[n];
        for(int i = 0; i < n; i++){
            arr[i] = Integer.parseInt(input[i]);
        }

        if(n == 1){
            System.out.println(0);
            return;
        }

        if(arr[0] == 0){
            System.out.println(-1);
            return;
        }

        int maxReach = arr[0];
        int steps = arr[0];
        int jumps = 1;

        for(int i = 1; i < n; i++){

            if(i == n - 1){
                System.out.println(jumps);
                return;
            }

            maxReach = Math.max(maxReach, i + arr[i]);
            steps--;

            if(steps == 0){
                jumps++;

                if(i >= maxReach){
                    System.out.println(-1);
                    return;
                }

                steps = maxReach - i;
            }
        }
    }
}
```

---

## 🧪 Sample Input

```text id="input1"
2 3 1 1 4
```

---

## ✅ Sample Output

```text id="output1"
2
```

---

## 🧠 Explanation

```text id="exp1"
Index 0 → jump to index 1  
Index 1 → jump to last  
Total jumps = 2
```

---

## 📊 Complexity Analysis

* Time Complexity: **O(n)**
* Space Complexity: **O(1)**

---

## 🔥 Key Points

* Greedy approach (not DP)
* Track max reachable index
* Count jumps only when needed

---

## ⚠️ Edge Cases

```text id="edge1"
[0, ...] → cannot move → -1  
[1 element] → 0 jumps  
```

---

## ⚡ One-line memory trick

```text id="mem1"
Expand reach, jump only when steps finish
```
