# 20. Find Maximum Product Subarray in an Array

## 📌 Problem Statement

Write a program to find the **maximum product of a contiguous subarray** in a given array.

---

## 📝 Input Format

* A single line containing space-separated integers (array elements)

---

## 📤 Output Format

* Print the maximum product of any contiguous subarray

---

## 💡 Approach (Explanation)

Unlike sum problems, product can change sign due to negative numbers.
So we track:

* `maxProd` → maximum product ending at current index
* `minProd` → minimum product ending at current index (important for negatives)

### Steps:

1. Initialize:

   ```text
   maxProd = arr[0]
   minProd = arr[0]
   result = arr[0]
   ```
2. Traverse array from index 1:

   * If current number is negative → swap `maxProd` and `minProd`
   * Update:

     ```text
     maxProd = max(current, maxProd * current)
     minProd = min(current, minProd * current)
     ```
   * Update result:

     ```text
     result = max(result, maxProd)
     ```
3. Print result

---

## 💻 Java Code

```java id="max-product-subarray"
import java.util.*;

class Main {
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);

        String[] input = sc.nextLine().split("\\s+");
        int[] arr = new int[input.length];

        for(int i = 0; i < input.length; i++){
            arr[i] = Integer.parseInt(input[i]);
        }

        int maxProd = arr[0];
        int minProd = arr[0];
        int result = arr[0];

        for(int i = 1; i < arr.length; i++){
            int curr = arr[i];

            if(curr < 0){
                int temp = maxProd;
                maxProd = minProd;
                minProd = temp;
            }

            maxProd = Math.max(curr, maxProd * curr);
            minProd = Math.min(curr, minProd * curr);

            result = Math.max(result, maxProd);
        }

        System.out.println(result);
    }
}
```

---

## 🧪 Sample Input

```text id="input1"
2 3 -2 4
```

---

## ✅ Sample Output

```text id="output1"
6
```

---

## 🧠 Explanation

Subarrays:

```text
[2] = 2
[2,3] = 6
[2,3,-2] = -12
[3,-2,4] = -24
[4] = 4
```

👉 Maximum = **6**

---

## 📊 Complexity Analysis

* Time Complexity: **O(n)**
* Space Complexity: **O(1)**

---

## 🔥 Key Points

* Track both max and min due to negative numbers
* Negative × negative = positive (important trick)
* Single pass solution
