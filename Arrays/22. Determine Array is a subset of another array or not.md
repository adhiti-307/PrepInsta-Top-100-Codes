# 22. Determine if One Array is a Subset of Another

## 📌 Problem Statement

Write a program to check whether one array is a **subset** of another array.

An array **B** is a subset of array **A** if **all elements of B are present in A**.

---

## 📝 Input Format

* First line: Space-separated integers (array A)
* Second line: Space-separated integers (array B)

---

## 📤 Output Format

* Print `Subset` if B is a subset of A
* Otherwise print `Not a Subset`

---

## 💡 Approach (Explanation)

1. Read both arrays
2. Store elements of array **A** in a `HashSet`
3. Traverse array **B**:

   * If any element of B is not present in the set → not a subset
4. If all elements are found → subset

---

## 💻 Java Code (Using HashSet)

```java id="subset-array"
import java.util.*;

class Main {
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);

        String[] inputA = sc.nextLine().split("\\s+");
        String[] inputB = sc.nextLine().split("\\s+");

        HashSet<Integer> set = new HashSet<>();

        for(String s : inputA){
            set.add(Integer.parseInt(s));
        }

        boolean isSubset = true;

        for(String s : inputB){
            int num = Integer.parseInt(s);
            if(!set.contains(num)){
                isSubset = false;
                break;
            }
        }

        if(isSubset){
            System.out.println("Subset");
        } else {
            System.out.println("Not a Subset");
        }
    }
}
```
