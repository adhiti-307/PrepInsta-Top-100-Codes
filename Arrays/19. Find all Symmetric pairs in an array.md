# 19. Find All Symmetric Pairs in an Array

## 📌 Problem Statement

Write a program to find all **symmetric pairs** in a given array of pairs.

A pair `(a, b)` is symmetric if another pair `(b, a)` exists in the array.

---

## 📝 Input Format

* A single line containing space-separated pairs in the format `(a,b)`

---

## 📤 Output Format

* Print all symmetric pairs

---

## 💡 Approach (Explanation)

1. Read the input pairs
2. Use a `HashMap` to store pairs:

   * Key → first element
   * Value → second element
3. Traverse each pair `(a, b)`:

   * Check if `map` contains key `b` and its value is `a`
   * If yes → `(a, b)` is a symmetric pair
4. Otherwise, store `(a, b)` in the map
5. Print all such pairs

---

## 💻 Java Code

```java id="symmetric-pairs"
import java.util.*;

class Main {
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);

        String[] input = sc.nextLine().split("\\s+");

        HashMap<Integer, Integer> map = new HashMap<>();

        for(String pair : input){
            pair = pair.replace("(", "").replace(")", "");
            String[] parts = pair.split(",");

            int a = Integer.parseInt(parts[0]);
            int b = Integer.parseInt(parts[1]);

            if(map.containsKey(b) && map.get(b) == a){
                System.out.println("(" + a + ", " + b + ")");
            } else {
                map.put(a, b);
            }
        }
    }
}
```

---

## 🧪 Sample Input

```text id="input1"
(1,2) (3,4) (5,9) (4,3) (2,1)
```

---

## ✅ Sample Output

```text id="output1"
(4, 3)
(2, 1)
```

---

## 📊 Complexity Analysis

* Time Complexity: **O(n)**
* Space Complexity: **O(n)**

---

## 🔥 Key Points

* Use `HashMap` for quick lookup
* Check reverse pair existence
* Efficient single-pass solution
